# 第六章 天牢里的指挥使

## 刑架之上

黑暗。
没有尽头的黑暗。

肖山感觉自己像是被一只无形的巨手死死攥住，整个人被某种不可名状的高维力量从内部撕碎、折叠，再粗暴地塞回一具陌生的躯壳里。
那种感觉无法用简单的“疼痛”来定义——那是一种存在性剥离的极度眩晕，仿佛他的意识代码被强行扔进了一个底层的垃圾回收站，经历了无数次死循环。

他想嘶吼，但声带发不出一丝声音。
视网膜上的景象如同老旧CRT显示器般疯狂跳动着雪花，最后，伴随着令人牙酸的失真音，画面猛然定格。

**咔嚓——**

刺耳的金属闭合声在耳膜深处炸开。极其沉重、冰冷的触感从手腕和脚踝同时传来。
肖山猛地睁开眼，瞳孔在极度的惊恐中收缩成了针尖。

这里不是废弃工厂。
也不是任何他认知中的现代建筑。

昏黄的火把光芒在视野边缘狂乱跳动，将整个空间映照成一片血色与暗影交织的炼狱。空气中弥漫着令人作呕的霉臭，混合着铁锈和浓烈的血腥气——那是陈年的干涸血迹被新鲜血液反复覆盖后，特有的腐败味道。

肖山低下头。
他惊悚地发现，自己的双手被粗糙的玄铁重链死死吊起，呈“大”字型固定在两侧的黑色木制刑架上。铁链末端的镣铐内圈，布满了生锈的倒刺，已经深深扎入了皮肉。温热的鲜血正顺着苍白的手腕蜿蜒而下，滴落在地上。
他下意识地挣扎了一下。
铁链发出沉闷而绝望的撞击声。肩膀关节处瞬间传来撕裂般的剧痛——这是肌肉长时间承受人体自重后，几乎要脱臼的折磨。

他的双脚同样被沉重的精钢脚镣锁死，铁链的另一头焊死在布满暗红色血斑的青石地板上。

刑架。
重镣。
水牢。

这三个充满封建极权色彩的名词，像三把重锤砸进肖山的大脑。
他的呼吸瞬间变得急促，胸膛剧烈起伏，冷汗瞬间浸透了单薄的囚衣。他拼命回溯上一秒的记忆——废弃工厂、满天金色的乱码圣旨、那句“宣逆贼肖山进京”……

然后呢？
然后就是这里。没有过渡，没有加载界面。他被系统强行跨纬度传送了！

肖山咬紧牙关，强行用程序员的理性压制住生理上的恐惧。他费力地抬起沉重的头颅，快速观察周边的环境。

火把插在墙壁的黑铁兽首托架里，每隔三步一支。这是一条深不见底的地下廊道，两侧是粗粝冰冷的巨大岩石。地面铺着极不规则的青石板，缝隙里积满了发黑的血垢。
廊道两侧，排列着一扇扇生锈的玄铁栅栏门。门后的深邃黑暗中，偶尔闪过一双双如死灰般的眼睛，伴随着令人毛骨悚然的低语和锁链拖拽声。

但他不在牢房里。
他被钉在廊道正中央的一片空地上。他的正前方，摆着一张散发着浓烈血腥气的长条木桌。桌上陈列着各式各样令人毛骨悚然的刑具：带血的倒钩、烧得通红的烙铁、细长的竹签，以及几把齿口发黑的剥皮钳。

肖山的胃部不可控制地痉挛了一下。

但真正让他感到如坠冰窟的，是刑架周围站着的人。

十几个身穿猩红飞鱼服的彪形大汉，如铁塔般整齐列阵。他们腰悬狭长的绣春刀，眼神如饿狼般死死盯着他。飞鱼服上的金线在火光下闪烁着嗜血的暗芒。

锦衣卫。

这三个字在肖山脑海中浮现时，他感到了深深的荒谬。
他是架构师。他的世界是由0和1、服务器、引擎代码组成的。他绝不该出现在这种带有浓厚大明极权色彩的封建刑房里！
但这真实的痛楚、粘稠的血腥味，无一不在警告他：这是系统的底层。这是那道“金色圣旨”背后的实体世界。

就在此时。
廊道的最深处，传来了一阵极其沉重、且极具压迫感的脚步声。

那脚步声不急不缓，但每一次靴底与青石板的碰撞，都像是一记重锤，精准地砸在所有人的心包上。带着一种不容直视的绝对霸权。

肖山的心跳，不自觉地被那脚步的频率所同化。

他震惊地看到，那两排原本气焰嚣张的锦衣卫，在听到脚步声的瞬间，齐刷刷地绷直了脊背。每一个人的眼中都涌现出极度的敬畏，甚至是——恐惧。

空气仿佛被抽干了。
一种无法用语言描述的庞大威压，如同实质化的海啸，从通道深处滚滚碾来。那不是物理层面的重力，而是一种更高维度的精神碾压！

肖山的双腿开始不受控制地剧烈战栗。
膝盖骨发出不堪重负的喀嚓声。潜意识在疯狂尖叫：跪下！跪下才能活命！

但他没有跪。
他死死咬碎了嘴唇，凭借着一种近乎执拗的骨气，用被吊在半空的双臂死命支撑着身体，不让自己瘫软下去。

“轰——！”
尽头的玄铁重门被猛然推开，狠狠撞在石壁上，发出震耳欲聋的轰鸣。

## 指挥使大人

火光剧烈摇曳，几乎被来人的气场生生压灭。

一道修长的身影，从黑暗中踏出。
那是一件纯粹的血红色大氅，如同一片翻涌的血海披挂在肩头。大氅之下，是一身剪裁极度修身的玄黑色蟒袍，腰束镶玉暗金宽带。他的左手，随意地按在一柄漆黑的狭长腰刀的刀柄上。刀鞘古朴，透着令人心悸的死气。

随着他的走近，那张脸终于在火光中无所遁形。

那是一张年轻到了极点，却又冷峻到了极点的脸。
五官深邃锋利，如同用最冷的冰刃雕琢而成。眉骨极高，眉锋如剑。皮肤呈现出一种久不见天日的病态苍白。
但最让人胆寒的，是他的眼睛。

那是一双漆黑如墨的眼眸。
没有愤怒，没有怜悯，没有任何属于人类的情感波动。就像是深渊底部死寂了千年的寒潭，又像是一台被彻底格式化、只剩下执行程序的机器。

在他出现的瞬间，整个天牢的温度仿佛暴跌至冰点。

肖山的目光与他对撞的刹那，脑子里“嗡”的一声，所有的思维都在瞬间宕机。

沈策。
是沈策。

可眼前这个穿着蟒袍、散发着滔天杀意的极权者，怎么可能是那个在末日废墟中与他并肩作战的沉默学长？！

肖山记忆里的沈策，是底层代码里的异常节点，是被系统标记为BUG的背叛者，是那个挡在他身前，对他说“滚出去”的男人。

而眼前这个人。
他身上散发出的上位者威压，简直就像是这个残忍系统的最高代言人。

披着血色大氅的沈策，步伐沉稳。他走到距离刑架三米的位置站定。那双毫无温度的眼睛，居高临下地扫过肖山。
那眼神，就像在审视一串毫无价值的垃圾代码。

而在沈策的视角里，事情同样诡异。
他握着刀柄的手指，在极细微地发力。并非因为愤怒，而是出于某种极其罕见的本能警觉。眼前这个伤痕累累的囚犯，身上散发着一种强烈的“违和感”。这种感觉超出了他常年审讯死囚的经验范畴，仿佛是一段不属于这个时代的乱码，被强行楔入了他的认知系统。

“唰——！”
两排如狼似虎的锦衣卫，在同一瞬间单膝轰然跪地。铠甲剧烈碰撞的金属声，在地下长廊里汇聚成令人颤栗的声浪。

“卑职，参见指挥使大人！”

锦衣卫指挥使。
三品大员。大乾王朝最恐怖的特务头子。

这几个头衔如惊雷般在肖山脑中炸响，彻底粉碎了他最后一丝侥幸。
在这个所谓的“大乾副本”里，沈策的身份被系统重置了！他不再是那个拥有代码翅膀的同行者，而是手握生杀大权的NPC暴君！
而自己，是被他踩在脚底的阶下囚。

沈策没有理会跪满一地的下属。
他甚至没有下令平身。他只是迈着那足以踩碎人心的步伐，一步，两步，径直走到肖山的面前。

那股窒息的压迫感达到了顶峰。
肖山浑身被冷汗浸透，双腿抖得像筛糠，但他依然死死咬着牙，倔强地抬着头，死死盯着那张熟悉又陌生的脸。

沈策停在距离他不到半米的地方。
火光在他的黑眸中跳跃，却点不燃一丝温度。

整个天牢陷入了死寂。只有远处偶尔传来的惨叫，和火把油脂的燃烧声。

终于，沈策开口了。
他的声音低沉、沙哑，如同极其名贵的大提琴在演奏挽歌。语调没有一丝起伏，但这极致的平静，却比任何严刑拷打都更具有穿透力。

“名字。”

两个字。如同两把冰冷的冰锥，直刺肖山的鼓膜。

肖山干咽了一口唾沫。喉咙里弥漫着血腥味，他用尽全身力气，从牙缝里挤出两个字：“肖……山。”

沈策的睫毛都没有颤动分毫。
“籍贯。”

肖山僵住了。
籍贯？在这个不知名的大乾王朝，他该怎么回答？北京朝阳区？还是废土避难所？任何一个不符合这个世界逻辑的回答，都可能立刻触发“抹杀机制”。

见他沉默，沈策的眼神依旧没有波澜，只是微微抬了一下那线条冷硬的下颌。

这是一个极其微小的信号。
但跪在地上的锦衣卫中，立刻有一名百户如同得到指令的猎犬般弹起，几步冲到桌前，一把抓起一根烧得通红、滋滋作响的烙铁！

“大人问话，你敢装聋？！”百户厉声暴喝，手中的烙铁直接朝着肖山的胸膛按去！

滚滚热浪扑面而来！肖山的瞳孔缩成了一个点。

“退下。”

低沉的声音响起。没有任何声嘶力竭，却带着绝对的统治力。
百户的动作瞬间定格在半空，烙铁距离肖山的皮肤仅剩寸许。他甚至不敢有丝毫犹豫，立刻收起烙铁，弓着身子退了回去。

沈策向前逼近了半步。
现在，他与肖山之间，几乎呼吸可闻。

肖山能看清他深邃的眼窝，甚至能闻到他身上那种混杂着冰冷松柏香与陈年血腥味的独特气息。

“本官再问一次。”沈策盯着他的眼睛，“你的籍贯。”

肖山的大脑在疯狂运算。
既然不能说真话，编瞎话又极易被这种特务头子拆穿。与其在逻辑的边缘试探，不如直接抛出死循环。

他盯着沈策的眼睛，沙哑地开口：
“我不知道。”

此言一出，周围的空气仿佛都凝固了。
几个锦衣卫的眼中露出极度惊骇的神色——从来没有人，敢在指挥使大人的面前用这种语气说话。

沈策的眼眸，终于微微眯起。
那是一个极度危险的信号。但在那一瞬间，肖山敏锐地捕捉到了他眼底深处的一抹异色——不是被冒犯的杀意，而是一种……极其深沉的探究。

就像是冰冷的水面，投下了一颗不知来历的石子。

“不知道？”沈策咀嚼着这三个字，语气中带上了一丝令人胆寒的玩味。

他缓缓转过身，背对着肖山，面向那些跪地的锦衣卫。
“全都滚出去。”

冰冷的五个字，让所有锦衣卫如蒙大赦。他们齐刷刷地叩首，连多余的眼神都不敢有，迅速起身，如退潮般鱼贯而出。

“哐当——！”
随着最后一人离开，沉重的玄铁大门被死死关闭。

阴森的天牢里，只剩下被吊在刑架上的肖山，和那位不可一世的指挥使大人。

沈策转过身，缓缓走向长条木桌。
他修长的手指在那些血迹斑斑的刑具上轻轻滑过，最终，拿起了一把短小精悍的剥皮剔骨刀。
刀刃在火光下闪过一道冷厉的弧光。

“本官诏狱里，死过很多人。”沈策看着刀锋，声音平淡如水，“有人嘴硬，上了夹棍就服了。有人嘴软，看一眼烙铁就尿了。像你这种……一问三不知的，本官一般会先剥下他一块皮，看看底下的肉，是不是也一样硬。”

他提着刀，走回肖山面前。

“但你，一定会招。”

肖山的心脏狂跳，但他强迫自己直视沈策的眼睛：“你想听什么？”

沈策举起两根修长的手指。
“高维妖术。”

这四个字，如同在肖山脑中引爆了一颗EMP炸弹。

沈策盯着他的表情，冷冷地陈述：“三天前，京师上空天幕撕裂，生出乱码异象。钦天监奏报，有逆贼动用‘高维妖术’试图篡改国运。圣上震怒，下令诏狱彻查天下。”
他用刀脊轻轻拍了拍肖山的脸颊，冰冷刺骨。
“而在那个坐标坠落的活口，只有你一个。”

肖山如遭雷击。
三天前？！他明明才刚刚被“圣旨”传送过来不到十分钟！这里的时间流速和之前的沙盒完全不同！
高维妖术……乱码异象……那分明就是他在废弃工厂利用BUG引发的系统崩溃！

“本官不在乎你用了什么妖术，也不在乎你从哪个位面来。”沈策将刀尖抵在肖山颈部的动脉上，微微施力。
一颗血珠渗了出来。

“本官只问最后一遍。”
沈策的眼神，充满了纯粹的杀戮意志。
“谁，派你来的。”

## 命运的红线

刀锋冰冷，动脉的跳动在刀尖下清晰可辨。

肖山的大脑在超频运转。
他现在终于明白了当前的死局——系统不仅重置了沈策的身份，还给他这个玩家植入了一个“阴谋论”的背景。在沈策（或者说大乾系统）的逻辑里，他肖山是一个被某个敌对势力派来搞破坏的间谍。

如果他说不知道，沈策绝对会毫不犹豫地切开他的喉管。
如果他说出真相——说“其实我们是校友，你是个BUG，这个世界是个系统”——在这个封建副本里，沈策只会把他当成失心疯，然后继续动刑。

他必须在这个沙盒的逻辑框架内，给出一个极其精妙的解释。
也就是——顺水推舟，编一个完美的谎言。

“大人说的幕后主使，我真的不知道。”肖山无视了脖子上的刀锋，直勾勾地盯着沈策，“但我知道一件事……大人，并不想杀我。”

沈策握刀的手，纹丝不动。但眼底的光，微微闪烁了一下。

“如果大人想要我的命，刚才那块烙铁就已经按在我胸口了。”肖山语速加快，像是在编织一张自我拯救的罗网，“大人支走所有人，不仅是不想让人听到我的供词，更是因为……大人心里，也有疑虑。”

沈策冷笑一声：“你在教本官做事？”

“我在陈述逻辑。”肖山目光灼灼，“大人说天幕撕裂是三天前。可如果，那个裂缝不是为了示威，而是为了……丢弃呢？”

沈策的眉头，微不可察地皱了一下。

“假设，幕后之人用了所谓的‘高维妖术’，只是为了打开一个通道，把我像个垃圾一样扔过来。我之所以什么都不知道，因为我就是个弃子！他们怕我知道得太多，所以清空了我的记忆。因为一个什么都不知道的白痴，就算落在鼎鼎大名的指挥使手里，也吐不出任何有价值的情报。”
肖山死死盯着沈策那张冰冷的脸，抛出了最后的筹码：
“但正因为我什么都不知道，我本身，就是最大的线索。杀了我，你就永远查不出天幕撕裂的真相。”

死寂。
死一般的沉寂在天牢中蔓延。

不知过了多久。
沈策笑了。

那是一个令人毛骨悚然的笑。没有温度，只有残忍的清醒。

“你的诡辩很精彩。”沈策收敛了笑意，声音再次降至冰点，“甚至差一点就说服了本官。”
他握紧了刀柄。
“但锦衣卫办案，向来宁可错杀三千，绝不放过一个。留着一个变数的风险，远大于杀掉他的收益。”

**铮——！**

寒芒一闪！
沈策竟然没有任何征兆地，直接挥刀斩下！

那速度太快了，快到超越了人类视神经的捕捉极限。肖山甚至没有感觉到恐惧，只看到一道刺目的银线在眼前炸开！

“死定了！”

**“喀嚓！喀嚓！”**
两声极其清脆的金属断裂声。

意想之中的身首异处没有发生。
捆绑在肖山手腕上的两条粗壮玄铁链，被这不可思议的一刀，精准地从中斩断！

失去悬吊力量的瞬间，肖山浑身的力气早已透支。他整个人如同一滩烂泥般，直挺挺地向前栽倒。

而在他正前方，站着沈策。
两人之间的距离，不足半米。

在倾倒的瞬间，肖山出于求生的本能，双手在空中胡乱地挥舞抓挠。
“啪。”

他的右手，死死地抓住了沈策垂在身侧的左手。

他抓住了沈策修长的手指。
冰冷。苍白。骨节分明。
在沈策的掌心处，有一道粗糙凸起的旧日刀疤，死死地硌在了肖山的掌纹上。

时间，在这一微秒，轰然凝固。

当两人的肌肤产生物理接触的那个刹那！
并非比喻，而是真正的物理法则崩溃！

肖山感觉整个天牢的重力瞬间颠倒。他的视网膜上，周遭的火光、刑架、石墙，全部扭曲成了疯狂拉伸的像素线条！
一种极其宏大、古老、且充斥着数据乱流的轰鸣声，在两人的脑海深处同时炸响！

就像是两段被强行斩断的命运，在接触的瞬间，突破了位面的封锁，产生了极其狂暴的共鸣！

沈策的身体剧烈地颤抖了一下。
那张永远冷酷如冰的面具，在这一刻彻底碎裂。他的瞳孔急剧放大，漆黑的眼眸中翻涌起惊涛骇浪。
某种被系统强行抹除的记忆碎片，如尖锥般刺入他的大脑。

而在肖山的视野中，世界发生了畸变。
他震惊地看到。
在沈策那苍白修长的脖颈上，在黑色蟒袍衣领的遮掩下，缓缓浮现出了一条极其纤细的、散发着刺目红光的纹路！

那是一条红线。
如同用最高权限的代码绘制而成，绕颈一周，在黑暗中如同脉搏般跳动。

与此同时，肖山的脖子上，传来了一阵如烙铁烫伤般的剧痛！
他能清晰地感觉到，自己的脖颈处，也浮现出了同样的一条红线！

两条红线。
跨越了时空，跨越了系统的重置，在黑暗的天牢中，产生了极其诡异的高维共振。

## 甲字七号

“砰！”

肖山重重地摔在青石板上。断裂的铁链砸出沉闷的回音。掌心的触感消失了，那种天旋地转的代码共振也随之戛然而止。

他剧烈地喘息着，冷汗早已将他湿透。
他艰难地抬起头，仰望着站在面前的那个男人。

沈策依旧保持着那个站立的姿势。但他握刀的手背上，青筋暴突。那双漆黑的眸子里，冰冷的杀意已经荡然无存，取而代之的，是一种极度混乱的迷茫与挣扎。

就像是一个觉醒了自我意识的NPC，正在与底层格式化协议进行着残酷的撕扯。

沈策缓缓低下头。
这是他进入天牢以来，第一次没有用那种居高临下的冰冷目光，而是带着一种不可置信的审视，看着地上的肖山。

“你……”沈策的嗓音干哑得可怕，仿佛每吐出一个字，声带都在流血，“你刚才……感觉到了什么？”

肖山死死咬着牙，忍着手腕的剧痛，点了点头。
“红线。”肖山喘息着，盯着他的眼睛，“你脖子上的红线。”

沈策的身体猛地僵直。
他下意识地抬起左手，按住了自己的侧颈。那里，此刻已经恢复了冰冷的苍白，红线似乎已经重新隐没在皮肤之下。

但他们都知道，那是真实存在的。

长久的死寂。

沈策闭上了眼睛。当他再次睁开时，眼底的混乱被强行压制，重新覆盖上了一层属于指挥使的冰冷面具。但肖山清楚，那面具已经裂开了。

“本官今天，不杀你。”
沈策的声音恢复了冷漠，但语气中却多了一丝难以察觉的疲惫。

他转身，大步走向玄铁门。

“来人。”

铁门轰然开启，两名锦衣卫校尉如鬼魅般闪入。

“把他关进甲字七号诏狱。”沈策没有回头，血色大氅在火光下翻卷，“找大夫给他止血。没本官的腰牌，任何人不得审讯，不得探视。他若死了，你们全家陪葬。”

两名锦衣卫浑身一颤，眼中闪过极度的震惊。
对于一个涉嫌“高维妖术”的钦犯，不仅不杀，反而给予了最高级别的保护？但他们根本不敢多问半句。

“卑职遵命！”

沈策迈出牢门。他的背影在长长的廊道中渐行渐远，最终消失在无尽的黑暗之中。

肖山脱力地瘫倒在冰冷的地砖上。
他赢了。或者说，他找到了这个世界最大的BUG。

沈策。
无论系统如何重置他的身份，无论他变成了多么冷血的杀人机器，他们之间的某种底层关联——那条未知的红线——依然存在！

那是系统无法格式化的羁绊。

两名校尉粗鲁但克制地将肖山从地上架起，拖向了诏狱更深处。

甲字七号牢房。
相对干燥的稻草，一扇极其厚重的纯钢大门。随着锁舌咬合的金属声，肖山被彻底隔绝在了黑暗之中。

他靠坐在冰冷的石墙上，抬起自己的右手，借着微弱的光线，凝视着掌心。
在废土世界，那个沈策曾经用满是刀疤的手掌，拉住他，对他说：“我信你。”
在今天，在这个荒诞的大乾王朝，他用这只手，抓住了指挥使沈策。

“不管你被系统改写成了什么怪物……”肖山喃喃自语，眼神在黑暗中亮得惊人，“我都一定会把你偏离的命运，给拉回正轨。”

而在同一时间的镇抚司后堂。

沈策摒退了所有下属，独自一人站在摇曳的烛光前。
他缓缓解开领口的盘扣，扯下衣领，侧身看向面前的铜镜。

镜子里，修长苍白的脖颈上，干干净净，没有任何痕迹。
但他却伸出手，指腹用力地在颈动脉处摩擦，直到皮肤泛起红肿。

那条红线。
那是他从有记忆起，就深藏在潜意识里的梦魇。它不属于这个大乾王朝，它属于一个他无法触及、无法理解的高维深渊。

而今天，那个自称“江临”的死囚，那个满口胡言乱语的男人。
竟然触发了它。

沈策看着镜子里的自己，眼中翻涌着令人战栗的疯狂。
他猛地拔出绣春刀，“喀嚓”一声，将面前的铜镜一刀劈成两半！

“你……到底是谁。”

黑暗的诏狱深处，与灯火通明的镇抚司后堂。
两个截然不同空间的男人，在这一夜，注定无眠。
