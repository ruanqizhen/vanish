# 第十四章：天地共振与千丈剑气

## 雷劫的奥义

在这种真正的生死关头，肖山的大脑反而进入了一种极其诡异的、绝对冷静的超频状态。

越是接近死亡的绝对边缘，他的思维就越是清晰。这是他在之前那无数次的死亡循环中，被系统强行逼出来的一种潜意识肌肉记忆。恐惧虽然会让人精神崩溃，但同时，也会让人的计算能力飙升到极致。

他死死地盯着天空中，那道正在乌云深处迅速凝聚成型的纯黑色恐怖雷霆。

不管是刚才劈下的第一道金色雷霆，还是那试图剥离他记忆的银色法则锁链，他在它们劈落前的短暂蓄力阶段，都极其敏锐地察觉到了一个致命的细节：它们的能量波动，都呈现出了一种极其完美、极其规律的周期性。

从极度暗淡到极度耀眼，再从极度耀眼迅速衰减到极度暗淡。

那是一个极其完美、符合经典物理学定律的——正弦波（Sine Wave）。

`f(t) = A × sin(ωt + φ)`

“正弦波？”肖山那布满血丝的眼睛越瞪越大。

作为一个曾经在底层架构中摸爬滚打多年的顶尖程序员，他对这种基础的正弦波简直太熟悉了。从最基础的音频信号处理、到庞大的城市交流电网、再到复杂的光波传播模型，他就算闭着眼睛，也能在键盘上极其流畅地敲出正弦波的傅里叶变换核心公式。

但是，代表着天道最高意志的雷劫波形，怎么可能会是一条如此基础、如此符合现实物理定律的正弦曲线？

雷劫，那是天道下达的绝对指令，是世界在天地层面上执行的终极抹杀法则。这种本该玄之又玄的底层法则，居然他妈的也是建立在一个最基础的正弦波函数之上的！

“等等……”肖山的脑海中猛地闪过一道犹如撕裂黑夜的灵光，“如果这个破雷劫的能量波形是一个正弦波，那就意味着——它绝对拥有一个固定的振动频率。而只要有频率，就必然意味着——可以产生物理共振。”

共振。

这是现实世界物理学中最基础、也是最可怕的一个概念。当一个物理系统的固有振动频率，与外部施加的激励频率完全相同时，就会产生极其强烈的共振现象。这种现象如果失控，足以震塌一座极其坚固的跨海大桥；但如果被极其精准地利用起来，它就能成为一个完美的能量放大器，或者是一个无损的能量接收器。

他的目光猛地像猎鹰一般，死死地转向了下方——在那片尚未被毁灭之光彻底抹去的残破街道上，有一辆明黄色的外卖电动车正在疾驰，骑手正拼了命地试图逃离那不断逼近的崩溃边缘。

电动车那廉价的电机马达，正在超负荷运转，发出了一种极其高频刺耳的嗡鸣声。肖山猛地闭上眼睛，将自己全部的意识，毫无保留地沉入到体内那颗正在狂暴运转的金丹之中。他试图用刚刚觉醒、还极其生疏的灵力感知，去跨越千米的距离，捕捉那个马达的振动频率。

37.5 赫兹。

那是那辆电动车马达此刻真实的振动频率。

紧接着，他又极其快速地感知了第二辆、第三辆……在新城区的几条残破街道上，还有大概十几辆正在狂奔的外卖电动车。它们因为电机老化程度和负载的不同，频率各不相同——有 35 赫兹的，有 40 赫兹的，还有一辆改装过的，频率接近 50 赫兹。

那么，刚才劈向他的第一道金色雷劫，它的震荡频率到底是多少？

肖山的瞳孔在极度的震惊中，猛然收缩成了一个极细的黑点。

37.5 赫兹。

和刚才那第一辆被他感知到的外卖电动车的马达频率——完全、绝对地一致！分毫不差！

“老子懂了！”肖山突然在千米高空中发出了一阵极其张狂的大笑，殷红的鲜血顺着他的嘴角疯狂流下，“这个狗屁天道的雷劫频率，根本就不是什么神秘固定的参数！它是在每次劈落之前，从这片残破的现实沙盒里，临时进行的底层数据采样！这片虚假的天地为了强行生成这种高维度的雷劫，为了节省它本身那已经濒临崩溃的算力，竟然直接抓取了沙盒里现成的物理频率，来作为雷劫能量的运行基准波！”

他猛地抬起头，死死盯住头顶那道已经彻底成型、即将轰然劈落的纯黑色雷霆。他再次将体内的灵力催动到极限，去探查那道黑色雷霆内部正在剧烈波动的能量频率。

50 赫兹。

那是林江市这种庞大现代都市，其底层供电系统的标准交流电频率。

这是这座庞大城市的工业脉搏，是这里每一盏路灯、每一台空调、每一个电子设备都在每分每秒使用的基础频率。这个系统，为了彻底抹杀他，竟然丧心病狂地调用了整个林江市电网的总频率，来作为载体，承载这道最致命、最高级别的灭世指令。

“既然你这个破系统要搞底层数据采样……”肖山的眼中闪烁着一种近乎疯魔的璀璨光芒，“那老子，就能跟你搞同频共振！”

这就意味着，他可以在物理层面上，把雷劫所代表的“抹杀意志”和它本身蕴含的庞大“能量”彻底分离开来。抹杀意志是软件层的指令，而承载这道指令的雷电波则是硬件层的载波。只要他能将自己身体的物理共振频率，极其精准地调节到与雷劫正弦波完全匹配的程度，那么，那股足以摧毁一切的抹杀意志，就会因为完美的相位抵消而彻底失效。而剩下的那些极其纯粹、没有任何属性的雷电能量，就会像水流一样，被他的身体完全、无损地吸收。

说干就干，在生死面前，没有任何犹豫的余地。

肖山极其粗暴地调动体内那颗已经处于暴走边缘的金丹灵力，试图强行改变自己身体能量场的固有振动频率。

40 赫兹。45 赫兹。48 赫兹。

他咬紧牙关，在极度的痛苦中，将频率死死地锁定在了一个绝对的数值上——50 赫兹。

就在同一瞬间，天空中，那道代表着绝对抹杀、连光线都能吞噬的黑色雷霆，带着毁天灭地的恐怖气势，轰然劈落！

它带着足以摧毁一切的恐怖威势，瞬间吞噬了百米的空间，直击肖山的头顶。

但这一次，奇迹发生了。

当那道足以让人形神俱灭的黑色雷霆接触到肖山身体的瞬间，它并没有像之前那样，残忍地撕裂他的血肉，也没有强行覆盖他的意识。它反而像是一条找到了宣泄口的奔腾河流，百川归海一般，极其平缓、极其顺畅地，顺着他全身的经脉，疯狂地涌了进去！

“成了！”

肖山的心中爆发出了一阵极度的狂喜。

那股冰冷、冷酷的抹杀意志，在他那已经达成同频振荡的灵力护盾面前，就像是一片脆弱的雪花落入了滚烫的沸水之中，瞬间消融得无影无踪。而被剥离了意志的、极其纯粹的庞大能量，则被他体内那颗如同饿死鬼投胎般的金丹，极其贪婪地一口吞下。

但仅仅在下一秒，极其惨烈的乐极生悲就降临了。

那颗金丹在吸收了这股堪称恐怖的庞大力量后，它的自转速度在瞬间暴增了整整十倍！金丹超负荷运转所散发出的极其恐怖的热量，让肖山的身体仿佛变成了一个正在燃烧的火炉。他整个人变得通红无比，皮肤表面甚至开始向外蒸腾出肉眼可见的白色蒸汽，他体内的每一条经脉都被那股庞大的力量撑得几近爆裂。

“操……能量过载了……”

他成功地将雷劫的力量导入了金丹，但他却忽略了一个极其致命的物理现实——他现在这具区区凡人的脆弱经脉，根本就承受不住这么庞大的能量冲击。这就好比是一条原本已经干涸的纤细小溪，突然被硬生生地汇入了一条决堤的狂暴江河，随时都会彻底崩溃。

与此同时，天空中的那片劫云，仿佛也察觉到了这个该死异数的底层小动作。

浓墨般的乌云开始剧烈地翻滚、沸腾，雷劫的频率开始出现极其疯狂、毫无规律的跳变。它不再是那种平稳的 50 赫兹，而是开始在 35 赫兹到 100 赫兹之间进行极其狂乱的随机切换。天道就像是一个被彻底激怒的程序员，正在愤怒地、毫无章法地变换着攻击手段和底层参数。

“我靠，这破系统他妈的还会变频？！”

肖山忍不住破口大骂。系统在不断地变换着攻击的 BPM 节奏，而他只能像个极其狼狈的舞者，拼了命地去踩那个疯狂跳动的节拍。但他毕竟只是一个刚刚吞下金丹的修仙界菜鸟，他的神经反应速度，根本就跟不上天道这种作弊般的变化速度。

一道极其粗壮的紫色雷霆，以一个极其刁钻的角度，狠狠地擦着他的左肩膀劈了过去。他那刚长好不久的半个肩膀，瞬间再次化作了一片虚无的飞灰。

“呃啊——！”

肖山死死咬紧牙关，不顾一切地用金丹的力量去强行修复那恐怖的伤口，他的大脑因为剧痛已经接近崩溃的边缘。

如果降频，他就会被紧随其后的雷劫瞬间劈死。
如果不降频，他体内的经脉就会因为严重过载而彻底爆体而亡。

这是一个绝对无解的死局。

“他妈的，既然横竖都是个死，那老子死也要拉个垫背的……”肖山的眼神中，爆发出了一种极度疯狂、玉石俱焚的光芒，“既然老子的身体能吸收这股能量，那老子就一口气吸个够！你这破金丹不是喜欢能量吗？老子今天就一次性把你喂到吐！”

他做出了一个极其疯狂的决定——彻底放弃这场毫无胜算的频率追逐战。

他直接敞开了自己全身所有的经脉防御，将肉体的防御力，极其决绝地降到了最低点。

“来吧！”他扬起满是鲜血的脸庞，仰天发出一声不屈的长啸，“雷劫也好，狗屁天道也好，老子今天倒要看看，你他妈的到底能不能劈碎一个程序员的求生欲！”

他不再躲避，而是极其主动地，像一颗炮弹般，逆流而上，迎向了那漫天狂暴的灭世雷暴。

## 踏雷筑基

轰！轰！轰！

数十道蕴含着不同毁灭属性的恐怖雷霆，在同一时间，极其密集地轰然劈落。金色的天罚、银色的因果锁、黑色的夺舍、紫色的毁灭、血红色的审判——各种颜色的狂暴雷电交织成了一片令人绝望的雷电海洋，瞬间将肖山那渺小的身影彻底淹没。

肖山没有任何的抵挡。

狂暴的雷霆之力毫无阻碍地贯入他的体内，极其恐怖的高压电流瞬间充斥了他全身的每一条经脉、每一根血管。他的身体在半空中极其剧烈、不受控制地抽搐着，原本白皙的皮肤表面浮现出了密密麻麻、如同细小灵蛇般疯狂游走的蓝色电弧。

而在他的丹田最深处，那颗金丹已经疯狂运转到了一个几乎要解体的临界点。

它就像是一个永远也填不满的黑洞，极其贪婪、不顾一切地吞噬着那些涌入体内的庞大雷霆之力。那些带着毁灭气息的紫黑色雷光，在金丹那原本光滑的表面疯狂流转，然后被一股极其霸道的力量强行剥离、吸收。在极度高压的淬炼下，金丹的颜色开始发生一种根本性的质变——它从原本那种稍显暗淡的暗金色，以肉眼可见的速度，蜕变成了一种晶莹剔透、散发着无尽威压的紫金色。

肖山的血肉之躯，在那恐怖的雷霆冲击下，开始不断地碎裂、崩解，然后又在金丹那磅礴生命力的灌注下，极其粗暴地重组、再生，接着再次碎裂、再次重组。

每一次痛苦到了极点的碎裂，都是一场向死而生的涅槃。
每一次血肉模糊的重组，都是一次彻头彻尾的脱胎换骨。

他体内那些原本脆弱不堪的经脉，在这种极度残忍的反复淬炼下，变得越来越坚韧、越来越宽阔。那些原本细如发丝、只能勉强容纳极其微小灵力的细小经脉，被那股狂暴的能量，硬生生地拓宽成了奔涌不息的滔滔江河。

他的意识，在那种超越极限的剧痛与一种极其奇异、难以言喻的生命升华感中，不断地浮沉、拉扯。

他的身体，正在发生一种彻底超越凡人生物学界限的高维质变。他体内的每一个细胞都在疯狂进化，骨骼被雷火重塑得坚硬如铁，他的灵魂，正在这雷劫的洗礼中，进行着痛苦而壮丽的升维。

之前强行吞下那颗属于别人的金丹，简直就如同一个三岁的小孩，不知死活地偷拿了一个核弹的引爆器，随时都会把自己炸得粉身碎骨。但现在，借由这天道雷劫极其狂暴的能量淬炼，以及那高维降维打击带来的绝对生死压力，这颗原本极其排斥他的金丹，终于被他彻底、真正地炼化，完美地融入了他的四肢百骸之中。

他的修炼境界，被这股狂暴的力量，极其野蛮、却又无比扎实地强行夯实了。

“原来如此……”肖山悬浮在那片刺目的雷光中心，双眼紧闭，嘴角却勾起了一抹极其明悟的微笑。

在刚才那种生死交关的雷霆淬炼中，他仿佛透过那些狂暴的能量表象，看到了雷霆内部正在流动的、最底层的世界法则——也就是创造这个虚假世界的本源代码。那就像是一本突然在他面前翻开的无字天书，每一个闪烁的符文、每一道复杂的阵法、每一条无形的因果线，都在向他极其清晰地揭示着这个世界最真实的运转规律。

代码的本质，到底是什么？
是对现实物理世界的数字化抽象。是对极其复杂、繁琐逻辑的完美封装。

而剑道，在某种高维度的意义上，同样也是一种极其高级的抽象。
剑招，是构成程序的语法。
剑意，是驱动程序运行的核心算法。
剑道，则是整个世界运转的顶层架构设计。

肖山在那狂暴的雷霆淬炼中，极其不可思议地，将那些最底层的世界法则，直接转化成了他对于剑道最深刻的领悟。他不再是那个只能在阴暗的地下室里，苦逼地写着业务逻辑的底层码农。他现在，成了一个能够直接用“剑”来编写和篡改物理法则的，超级黑客。

“原来……剑本身就是一种法则。而剑气，就是这套法则最纯粹的执行意志。”

他缓缓地睁开眼睛，伸出了自己的右手，极其自然地并指为剑。

那是一只极其普通的手，白皙、修长，手指上甚至还带着常年敲击键盘留下的老茧，与任何一个普通人的手没有任何区别。

但是，就在他并拢的两根手指之上，却极其突兀地，凝聚出了一道光——那是一道纯粹到了极致、由无数极其微小、极其复杂的金色符文紧密交织而成的一道金光。

那道金光只有短短的三寸长，看起来就像是一根极其微不足道、随时都会熄灭的发光探针。

但就是这根只有三寸长的探针出现的那一瞬间，那片正在疯狂倾泻雷劫的恐怖乌云，竟然出现了极其诡异的一瞬停顿。这片天地的底层法则，似乎极其敏锐地感知到了一股极其致命的威胁，感知到了一个本不该存在的、掌握了底层逻辑的破局者，正在这片虚假的天地间诞生。

肖山看着指尖的那道金光，嘴角再次溢出了一丝温热的血丝，但他却极其畅快地笑了。

“老子这辈子，坐在电脑前，敲了几十万行的破代码……行行都是他妈的屎山，行行都是修不完的 BUG。”他的声音在狂风雷暴中，带着一种极其浓烈的自嘲与一种看破一切的傲然，“今天，老子要在这个破世界里，写一行最他妈干净、最没有 BUG 的代码。”

他并指如剑，极其沉稳地对准了头顶那片压抑到了极点的劫云，然后，缓缓地、毫无花哨地刺了出去。

在他那已经被雷霆彻底拓宽、强化的神经网络最深处，一段他从来没有在任何键盘上敲击过、却无比清晰的代码，极其自然地浮现了出来。正弦波的频率、雷劫的能量特征、金丹的因果连结、底层世界逻辑的破绽——所有的碎片信息，在这一刻，自动、完美地编译成了一段绝对可执行的核心逻辑：

```java
if (thunder.frequency == self.resonance) {
    thunder.energy.absorb();
    thunder.instruction.bypass();
    return swordBeam(thunder.energy * causalLines);
}
```

他刺出的这一剑，没有任何花哨、繁复的技巧。
也没有任何惊天动地、声势浩大的能量威势。
那就是简简单单、极其质朴的一刺，其动作之简单，简直就如同一个程序员，在键盘上极其随意地重重敲下了一个回车键。

但是，就是这个看似微不足道的回车，却极其致命地，触动了整个沙盒世界最底层的运转逻辑。

剑尖刺出的那一瞬间，金光，轰然暴涨！

那原本只有三寸长的金光，在脱离他手指的瞬间，迎风疯狂暴涨。三寸、三尺、三丈、三十丈、三百丈……在不到一秒钟的时间里，竟然暴涨到了恐怖的三千丈！

一道长达千丈、璀璨夺目到了极点的金色剑气，犹如一根支撑天地的擎天巨柱，横贯了整个林江市的夜空！

## 天幕裂痕

那道千丈剑气，带着一种一往无前、斩断一切因果的恐怖决绝，逆天而上。

它首先极其霸道地，将那道正在疯狂倾泻的黑色雷柱，从正中间完美地劈成了两半，那干脆利落的程度，简直就像是一把烧红的热刀切过了一块柔软的黄油。被劈开的雷霆力量彻底失去了目标，向着两侧极其狼狈地崩溃、宣泄，最终轰击在林江市郊外的荒野上，引发了一阵极其剧烈的地质震动。

但那道剑气的去势，没有丝毫的减弱，它继续以一种不可阻挡的姿态向上斩去。

它极其狂暴地劈开了那片浓郁的劫云，劈开了那些在乌云中疯狂流转、试图重组的底层符文网络，劈开了这片天地为了抹杀异数而强行施加的最后一道逻辑屏障。

那漫天翻滚的恐怖雷云，竟然被这极其惊艳的一剑，生生地劈作了两半！

整个紫黑色的天空，被这道剑气，极其粗暴地撕开了一条长达数公里的巨大缝隙。

一种极其奇异、璀璨的金色光芒，顺着那条巨大的缝隙，如瀑布般倾泻了下来。那绝对不是属于这个世界的阳光，而是一种更为古老、更为神秘、更为纯粹的光——那是来自这个封闭世界之外的光线，或者说，那是属于更高维度空间的辐射。

原本不可一世的雷云，在这道恐怖的剑气下，迅速碎裂成了无数黑色的飞灰，在夜风中凄凉地消散。

那个极其冰冷的系统机械音，在肖山的脑海中最后一次响起，只是这一次，那声音中带着一种极其严重、濒临崩溃的系统卡顿：“检测到……异……异数……终极抹杀指令执行……失……败……底层边界结界……遭受致命受损……”

肖山静静地悬浮在那千米的高空之上。他浑身浴血，那件标志性的外卖制服早已经在雷劫中破碎成了丝缕，但他那残破的躯体里，却充盈着一种前所未有、极其浩瀚的恐怖力量。

他，筑基了。

凭借着一颗从别人那里强行抢来的金丹，以及一场被他抓住了底层逻辑漏洞的灭世雷劫，他一个手无缚鸡之力的凡人，竟然硬生生地跨越了凡人与修真者之间那道不可逾越的界限，完成了一次不可思议的生命升维。

但他现在根本就来不及去细细体会体内那股全新的力量。
他那双布满血丝的眼睛，被天空中那道被他一剑劈开的巨大裂缝，死死地吸引住了。

那道裂缝的尽头，并不是他预想中浩瀚无垠的宇宙星空。
而是一片极其辽阔、表面流转着无数玄奥符文的半透明“天幕”。那层天幕就像是一层极其坚韧的薄膜，死死地包裹着林江市所在的这个沙盒世界。而此刻，这层看似坚不可摧的天幕，被他那惊天动地的一剑，极其暴力地划开了一道触目惊心的口子。

透过那道裂开的口子，他看到了，另一个完全不同的世界。

在一片极其壮丽、翻滚的云海之上，一座巍峨到了极点的远古仙宫，正在极其惨烈地崩塌。无数巨大的白玉柱轰然坠落，燃烧着的金色符箓犹如漫天的火雨般纷纷扬扬，无数道流光从天宫的深处向四面八方惊恐地逃散，那画面，就像是神话传说中的诸神黄昏，惨烈而壮观。

而在那座仙宫崩塌的最中心位置，静静地伫立着一道身影。

那是一个白衣胜雪的女子。她那如瀑的长发在灵气风暴中狂乱地飞舞，她的手中，紧紧地握着一把散发着青色光芒的古剑，正孤独而决绝地，面对着天穹之上，那无数道正在疯狂降下的高维毁灭光束。

她的面容极其模糊，仿佛被一层极其神秘的雾气所笼罩，但肖山却极其清晰地感觉到了她的目光——那道目光穿透了崩塌的仙宫，穿透了遥远的维度阻隔，穿透了那道被劈开的天幕裂痕，在无尽的虚空中，与他那充满震惊的目光，极其玄妙地交汇在了一起。

紧接着，他听到了一个声音。
那个声音没有通过任何物理介质传播，而是直接在他的灵魂最深处，极其清晰地回响了起来。

“找到我。”

只有简简单单的三个字。但那声音里，却带着一种穿越了千万年时光的沧桑感，以及一丝掩饰得极好，但依然难以察觉的极度疲惫。

还没等肖山做出任何反应，那道被撕裂的天幕，已经开始了极其迅速的自我修复。无数金色的底层系统光芒如同活物一般疯狂地蠕动、交织，以一种肉眼可见的恐怖速度，修补着那道巨大的裂缝。崩塌的仙宫、那个孤独的白衣女子、翻滚的云海……这些极其震撼的画面，在他的视线中急速收缩、变得模糊，最终，彻底消失不见。

林江市的天空，再次极其严密地闭合了起来。

但在天幕彻底闭合的最后一瞬间，有几枚极其微小、晶莹剔透的碎片，极其艰难地从那层天幕上剥落了下来，化作了几道极其微弱的流星，悄无声息地坠入了林江市那片废墟的几个角落之中。那是几枚夹带着另一个高维世界气息的数据碎片。

肖山深吸了一大口充满焦臭味的空气，正准备控制着体内那股全新的筑基灵力降落下去，去寻找那些神秘的碎片。

就在这时，他的胸口处，突然传来了一阵极其剧烈的灼热感。
那是沈策留给他的那块，没有任何指针的怀表！

那块怀表仿佛受到了某种高维力量的召唤，极其突兀地从他的衣服里飘了出来，静静地悬停在他胸前一尺的地方。那原本就布满了裂纹的银色金属表面，此刻正散发着一种极其深邃、迷人的深蓝色幽光。那股力量波动的频率，与之前沈策在地下室里强行撕裂空间时使用的力量，简直如出一辙。

“等等……”肖山的大脑迅速运转，他瞬间意识到了这块怀表的真正用途。

这东西，绝对不仅仅只是一个用来短距离逃生的工具。这块怀表，是一件极其罕见的、能够直接跨越世界维度的“高维法器”！

怀表上的深蓝色光芒骤然爆发，在半空中极其迅速地形成了一个不断旋转的蓝色空间漩涡。
一股庞大到根本无法抗拒的吸力，直接越过了肉体，极其霸道地作用在了肖山的灵魂之上。

“不！我他妈的还没找回沈策消散在这个世界里的代码碎片！”肖山在半空中拼命地挣扎，试图用体内那股刚获得的筑基灵力，去强行对抗那个空间漩涡的恐怖吸力。

但那股力量，带有极其高级的不容抗拒的天地法则属性。那是沈策用自己彻底消亡作为惨痛代价，为他换来的一次单向维度传送，这股力量，绝对不可逆转。

蓝光瞬间暴涨，将肖山那正在挣扎的身影，彻底吞没。

林江市那满目疮痍的夜空，终于彻底恢复了死寂。没有了代表系统杀毒程序的狂暴雷劫，没有了那种毫无感情的高维毁灭之光，也没有了那个手握千丈剑气、一剑斩开天幕的疯狂程序员。
只剩下这座千疮百孔、伤痕累累的城市，在冰冷的夜风中，发出极其微弱的无声哀鸣。

……

不知跨越了多少个光怪陆离的维度世界之外。

肖山感觉自己的意识在一条无尽的蓝色通道中以极其恐怖的速度穿梭。他的身体仿佛化作了一道没有重量的流光，周围的空间法则正在被不断地折叠、扭曲。

当那种极其难受的极致失重感终于彻底结束时。
他极其狼狈地，重重摔在了一片极其坚硬、冰冷的地面上。

“砰！”

他痛苦地捂着隐隐作痛、仿佛要裂开的脑袋，艰难地，缓缓抬起了头。

映入他那充血眼帘的，是一根足有百丈之高、表面盘绕着一条栩栩如生的五爪金龙的巨大纯金柱子。他的脚下，铺满了一块块光可鉴人、散发着微弱灵气的金砖。而在他的两侧，密密麻麻地站满了身穿华贵朝服、气息极其深不可测、犹如渊海一般的古代文武百官。

他发现自己，此刻正以一种极其屈辱的姿态，跪趴在一座极其巍峨、充满着无尽皇家威压的宫殿正中央。而他身上那件原本破破烂烂的明黄色外卖制服，不知道在何时，竟然已经被系统强行覆盖，变成了一件极其华贵、玄黑底色上用金线绣着獬豸图案的古代官袍。

在大殿正前方那高高在上的巨大高台上，一个身穿极其刺目的紫金蟒袍的中年男人，正居高临下地看着他，那目光，就像是两把万载寒冰雕刻而成的利刃，极其冰冷、残忍。

“御史大夫肖山，面圣不跪，御前失仪，按照大乾律法，你该当何罪！”
