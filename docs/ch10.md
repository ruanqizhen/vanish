# 第十章：死亡排序

## 快速排序，活人算法

黑暗与封闭，是滋生恐惧最肥沃的土壤。

巨大的宴会厅内，只有中央那块二十米的LED屏幕散发着血红色微光。几百名政商名流在经历最初的尖叫与砸门后，终于被更深层的恐惧压得噤了声。

空气弥漫起淡淡铁锈味。
屏幕上的血海翻涌，凝聚成一个巨大、模糊、没有五官的类人轮廓。完全由红色像素点拼凑的“厉鬼”，头部以扭曲角度，缓缓扫视下方发抖的人群。

“欢——迎——来——到——我——的——宴——会。”
电子合成音如指甲划过黑板，在三百六十度回荡。

肖山站在离主桌不远的地方，冷汗打湿后背。
他感觉脖颈处传来细微勒紧感。曾在地下室被强行挂上的那条透明红线，此刻隐隐发烫。
系统换了杀戮的皮肤，从修仙背景的“天劫”，变成都市背景的“怪谈”。但底层的抹杀机制，依然冷酷。

“沈策，”肖山压低声音，余光扫视周围，“这玩意儿要干什么？”
沈策不知何时捡起一把纯银餐刀，刀锋泛着冷意。他看着屏幕，眼底是死寂的冰冷：“点名。”

话音刚落，屏幕上的厉鬼缓缓抬起红色手臂，指向观众席最前方A区。
“第——一——位——”
声音拖拽着令人毛骨悚然的尾音。
“陈——国——辉。”

“啊！！！”
A区第三排，一个大腹便便的地产商发出杀猪般的惨叫。他从椅子上弹起，惊恐地看着双手——
皮肤表面出现密密麻麻的数字代码，皮肉如同烈火中的蜡块，迅速融化！
“救命！我不想死！救——”

凄厉惨叫仅持续了三秒。
三秒后，陈国辉彻底消失，原地只留下一滩腥臭的暗红血水，连昂贵西装都被蒸发。

全场死寂。
紧接着，更疯狂的尖叫爆发，有人试图拿椅子砸门，但门上瞬间腾起蓝色数据火焰，将触碰者直接烧成灰烬。

“谁也跑不掉。”肖山呼吸急促，强迫视线从血水移开，作为架构师的本能被强行激活。

这不是灵异事件，这是系统在执行代码。
陈国辉的死，是代码执行结果。

“第——二——位——”手指转向C区。
“李——秀——芬。”

肖山猛地转头。
C区角落，一个穿沈氏IT制服的短发女人捂着右臂跪倒。她的右臂数据化碎裂，露出幽蓝代码。
但，她没死！
融化进行到一半停止了，她死咬着牙，撑着地面，活了下来。
肖山认出她，是曾提醒他“小心红线”的安全工程师。她身上绝对有抵抗系统抹杀的手段。

没等深想，厉鬼已指向B区。
“第——三——位——张——伟。”

“老子跟你拼——”B区一个年轻富二代怒吼举起椅子，但话音未落，身体像被无形大手折叠捏碎，伴随骨裂声，化作第三滩血水。

肖山死死盯着三个人倒下的位置。
冷汗滑落。
陈国辉。李秀芬。张伟。
共同点？性别？年龄？身份？
肖山目光扫过椅子上的编号牌。

陈国辉：52岁，A区3排7座。
李秀芬：25岁，C区9排2座。
张伟：28岁，B区6排5座。

数据在肖山脑海飞速重组、碰撞。
作为顶级架构师，他太熟悉这种筛选模式了。

“点名顺序不对。”肖山咬着指甲，瞳孔震颤，“在数据库里随机杀人不需要指东打西。这不是随机，是……排序。”
他猛地抓住沈策手臂，力道惊人：“它在执行快速排序（Quick Sort）算法！”

沈策眉头微皱，紧握餐刀：“说人话。”

“它把我们在场所有人当成了数组元素！”肖山语速极快，“第一步，选取一个‘基准值’，就是那个52岁的陈国辉！然后，把剩下的分成两组——一边是小于陈国辉，一边是大于陈国辉！”

“它在干什么？”
“在整理‘数据库’！”肖山声音发抖，“李秀芬25岁，张伟28岁，都在陈国辉的‘左侧（小于基准组）’！厉鬼每次点名，就是从递归生成的子数组里挑人杀！”

“第——四——位——王——建——国。”声音再次响起。
A区另一侧，一个56岁老人哀嚎化作血水。

“看到了吗！”肖山指着血水，“王建国，56岁！他在陈国辉的‘右侧（大于基准组）’！这个排序机制是真的！”

“知道是排序有什么用？”沈策看着不断有人化作血水，眼神冷酷，“能阻止它？”
“不能阻止，但能预测！”肖山眼睛发亮，“只要算出排序键值，就能推断下一个死的是谁。在这个见鬼的规则里，预知顺序，就是预知死期！”

## 异常值

第五个人死了。
刘志强，45岁，B区2排4座。左侧数组。

第六个人死了。
林美凤，49岁，A区5排6座。左侧数组。

肖山的脑子快爆炸了。
不对。如果主键是年龄，逻辑出现严重断层。按照经典快排递归，左右子数组处理顺序出现悖论，王建国的位置对不上。

不是年龄。
不是座位号。
不是身价。

“姓氏……”肖山扫过六个人的名字，“陈（C）、李（L）、张（Z）、王（W）、刘（L）、林（L）。如果是按拼音首字母ASCII码值排序呢？”

就在这时，屏幕红光剧烈闪烁，发出刺耳蜂鸣。
厉鬼没有五官的头部，猛地转向肖山的方向。

“第——七——位——”

肖山心脏骤停。
屏幕上，缓缓浮现两个血淋淋大字：
**江。临。**

肖山，曾用名：江临。
这是他现实世界的真名！系统连他在大乾时空用的“肖山”这层皮都扒了！

**【10……】**
厉鬼头顶出现倒计时。
前六个人直接处刑，而他这里，出现了倒计时！

“江临？是你？”沈策一把拉住肖山，眼神凌厉。
“是我。”肖山感觉脖子被透明红线勒得快窒息了。

**【8……7……】**

肖山看着倒计时，大脑短暂空白。
完了。
算错了。
排第七个，拼音首字母J，完全不符合快排逻辑！厉鬼用的不是快排，或者复合主键复杂到无法在十秒内破解！

**【5……4……】**

沈策死抓肖山手臂，肌肉紧绷，银餐刀被握得变形。他盯着倒计时，眼神决绝——如果系统要杀肖山，他就算拼着神魂俱灭，也要把这屏幕连同宴会厅斩碎！

**【3……2……1……0。】**

归零。
肖山闭上眼。

一秒。
两秒。
三秒。

想象中的融化没来。
剧痛没来。
红线都没勒紧。

肖山猛地睁眼。
摸了摸身体。没少一块肉。
怎么回事？系统死机了？

“看屏幕。”沈策低沉声音响起。

肖山抬头。
屏幕上“江临”两个大字，像遇到严重逻辑错误，疯狂闪烁扭曲，化作乱码。
紧接着，乱码被强行抹除。

厉鬼再次开口，声音竟然带上一丝人性化的……愤怒与卡顿。
“第——七——位——”
“叶——婉——秋。”

角落里，一个中年女人惨叫化作血水。
真正的第七个人死了。而肖山，活了下来。

“它跳过我了？”肖山大口喘气。
“不，它没有跳过你。”
沈策目光落在肖山脖颈上，那根只有两人能感知的红线，正散发着不属于这个系统的暗紫光芒。

“你是‘异常值’。”
沈策看着肖山，眼底闪过明悟。
“在排序算法里，你被选中了。但你身上绑着另一套高维规则，两套规则产生了逻辑死锁。程序无法执行‘抹杀指令’，为防止算法崩溃，只能将你标记为‘异常报错’，强行跳过杀下一个人。”

肖山明白了。
木偶师说的“你已经被我牵上了线”，是变相的保命符！
只要红线在，他就是“不可判定数据”。系统分出胜负前，他死不了！

屏幕倒计时再次亮起。
这次，是**【60】**秒。

间歇时间拉长了。
但不意味着安全，随着人数减少，算法收敛速度会越来越快。当所有正常“数据”杀光，系统腾出所有算力处理他这个“异常值”时，就是死期。

就在这时，屏幕上的厉鬼，那张没有五官的脸上，嘴部裂开诡异缝隙。
用一种只有“异常值”能听懂的极低频率，吐出两个字：

“跑——吧。”

## 死局的出口

跑？
门窗全部封死、触碰就会被数据火焰烧成灰烬的宴会厅里，往哪跑？

肖山看着满地鲜血和崩溃的宾客，大脑高速运转。
不对。
既然是“电梯求生”规则，就一定有电梯！

“沈策，”肖山一把拉住沈策手腕，“规则说过，‘在酒店内找到隐藏电梯井’。大门出不去，说明电梯根本不在外面！”
“在这个大厅里？”沈策眯起眼睛扫视。

“对！既然是怪谈，一定遵循空间重构逻辑。”肖山在宴会厅穿梭，“电梯井……垂直空间……如果四周是死路，只能在……”

目光猛地落在正中央地板。
香槟塔倒塌的位置，暗红波斯地毯上，有一个用银线缝制的沈氏族徽。
六边形图案。
正好是电梯轿厢底盘形状！

“地毯下面！”肖山指着那。
沈策大步流星走过去。将纯银餐刀狠狠扎进地毯边缘缝隙，肌肉骤然暴起，“嘶啦”一声，沉重奢华的波斯地毯被硬生生撕开巨大口子！

地毯下方，根本不是大理石。
是一个深不见底、散发阴冷金属光泽的四方竖井！
竖井边缘，甚至挂着一条锈迹斑斑的钢缆，直通无尽黑暗。

“电梯井……”几个幸存者像抓住救命稻草，疯了一样扑来。
“找到了！出口在这！”

然而，没等靠近，屏幕倒计时归零。
“第——八——位——”
“周——建——明。”

扑在最前面的男人，惨叫都没发出，直接在电梯井边缘化作血水，顺着竖井滴落深渊。

“不能等了。”
肖山看着竖井下方令人窒息的黑暗，转头看向沈策。
“下面不知道通向哪里，也许是更深的规则地狱。”

沈策走到边缘，抓起锈迹斑斑的钢缆，用力扯了扯试探承重。
回头看着肖山，嘴角勾起狂傲冷笑。
“地狱又如何？”

“那就走着！”
肖山大笑一声，毫不犹豫跃入竖井，双手死抓钢缆，身体极速滑落。
沈策紧随其后，犹如黑色蝙蝠隐入黑暗。

身后，宴会厅大屏幕猩红光芒疯狂闪烁。
厉鬼脸庞扭曲，伴随惨叫与血水，这场疯狂点名依然在执行最后的数据清洗。

而肖山和沈策，已经进入了怪谈最深处。

**[第十章·完]**
