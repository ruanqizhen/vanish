# 第十五章：大乾金殿的锋芒

## 大梦初醒

在那条深蓝色的虚空通道彻底碎裂的刹那，肖山感觉自己的身体仿佛被无数种不同维度的高维力量同时疯狂地撕扯、极其残忍地解构，然后再以一种他完全无法理解的方式，被极其粗暴地强行重组。

那种疼痛，已经彻底超越了人类肉身能够感知和承受的物理极限——那感觉就像是有一个极其残忍的刽子手，拿着一把烧红的烙铁，正在他的灵魂最深处，一寸一寸地强行刻下新的身份烙印。他原本在现代都市里的记忆、身份标识，正在被另一套庞大、古老的数据极其霸道地强行覆盖、重写。他的视野中犹如走马灯一般疯狂闪烁过无数极其陌生的光影：极其广阔浩瀚的大乾帝国疆域图、密密麻麻如同繁星般的百官命格星盘、还有那座代表着绝对皇权的千年金銮殿所散发出的恐怖威压……这些极其庞大、冗余的信息，如同一场决堤的洪水，被系统极其蛮横地强行塞入了他的识海之中。

他，被系统像扔垃圾一样，极其随意地扔进了一个全新的世界沙盒。

砰！

那种令人作呕的失重感骤然消失，肖山从半空中直直地坠落，双膝极其重重地砸在了一块极其冰冷、坚硬的纯金地砖上，发出了一声沉闷的巨响。

原本庄严肃穆的大殿，瞬间变得鸦雀无声。

肖山极其艰难地撑着酸痛的双臂，缓缓地抬起了那颗仿佛要爆炸的头。映入他眼帘的，是一座极其巍峨、奢华到了极点的古代宫殿。那是一根根足有百丈之高、需要数十人合抱的巨大金柱，柱子上盘绕着仿佛拥有真实生命的五爪金龙，那一双双龙目之中，甚至还在极其隐秘地吞吐着一种淡金色的高维度灵气。大殿的穹顶极高，高到根本看不见尽头，只能看到那上面正在不断翻滚的金色云海，以及一个极其庞大、正散发着恐怖威压的星斗大阵在云海中缓缓沉浮。大殿两侧站立着的文武官员，每一个人的气息都犹如深不可测的深渊，他们身上散发出的灵压在空气中极其复杂地交织在一起，形成了一张让人几乎无法呼吸的无形巨网。

大乾帝国的权力核心，金銮殿。

而他肖山，此刻正以一种极其狼狈的姿态，跪趴在这个代表着整个沙盒世界最高权力中枢的正中央。

他的手指下意识地摸向了自己的胸口——隔着那层极其华贵的丝绸官袍，那块没有指针的神秘怀表，依然在向他传递着一丝熟悉的冰凉。可是，那个把怀表交给他的男人，那个总是冷着一张脸的沈策，已经彻底不在了。沈策用他自己的命，硬生生地撕开了一道维度裂缝，极其粗暴地把他送到了这个全新的世界。

肖山深吸了一大口充满了奇异檀香味的空气，极其果断地松开了握着怀表的手，转而极其自然地，握住了自己腰间悬挂着的那把黑色剑鞘。

当他的手握住剑鞘的那一瞬间，他整个人的眼神，彻底变了。从那个刚刚在末日雷劫中狼狈求生、死里逃生的底层外卖员，瞬间变成了一把已经出鞘、散发着刺骨寒意和决绝杀意的绝世冷锋。

“大胆！”

一声犹如惊雷般的暴喝，极其突兀地在这座死寂的大殿中轰然炸开。

肖山循着声音冷冷地望去。在那一群气焰滔天的文官之首，站着一个身穿极其刺目、用金线绣着张牙舞爪图案的紫金蟒袍的中年男人。那人面如冠玉，颌下留着极其漂亮的长髯，但那双眼睛，却如同两口深不见底的万载寒潭，透着一种视万物如蝼蚁的极致冰冷与傲慢。

那人周身环绕着极其浓郁的紫色气运，那是一品大员才能拥有的位阶威压，此刻那股威压浓郁得几乎已经化为了物理意义上的实质，在他的头顶上方，形成了一顶散发着恐怖压迫感的紫金华盖。而更让肖山感到一丝心悸的是，在这具看似文弱的身体里，竟然蕴含着极其恐怖的灵力波动——修为已至渡劫巅峰。那可是差一步就能白日飞升、彻底脱离凡人范畴的恐怖境界。

在这个大乾王朝的朝野上下，有资格穿这件紫金蟒袍的，只有一个人：当朝太师，赵无极。

当这个名字在他的脑海中浮现的瞬间，属于这个新身份的记忆如泉水般疯狂涌出：这是一个历经三朝的不倒翁元老，他门下的弟子故吏遍布整个大乾的朝野上下，甚至在民间有着“半朝赵家”的恐怖称号。此人表面上极其忠君爱国，实则早已经用极其卑劣的手段，将整个帝国的权力网络，极其严密地编织成了他个人的私有玩具和提款机。

但是，在刚刚跨越了生死雷劫、彻底觉醒了底层视觉的肖山眼中，这位高高在上的太师，其真实的系统身份，远远不止是一个简单的权臣那么简单。

肖山微微眯起了眼睛。他那双刚刚经历过雷劫淬炼、成功筑基的眼睛，极其轻易地穿透了这个沙盒世界虚伪的表象。

赵无极那隐藏在底层代码中的真实信息面板，极其清晰地在肖山的视网膜上展开：

【目标名称：赵无极】
【社会职位：太师（正一品，极度专权）】
【物理修为：渡劫期·九重天巅峰】
【系统气运值：9,999,999,999 / 10,000,000,000】
【底层身份：本沙盒世界线修正核心锚点】
【系统威胁等级：极其致命】
【底层备注说明：该单位承载了本世界高达 80% 的剧情权重，将其击杀，可强制破除部分系统因果锁定】

肖山彻底看清了。

这个高高在上的老家伙，根本就不是什么狗屁历史人物，他是这个沙盒世界里最大的一个“系统毒瘤”，是一个极其卑劣地伪装成人类的、用来控制世界走向的机制怪物！

“御史大夫肖山，面圣不跪，御前失仪，按照大乾律法，你该当何罪！”

赵无极的声音其实并不高，但那声音里，却极其诡异地带着一种仿佛天道法则般的恐怖压迫力。他吐出的每一个字，都像是一记极其沉重的震天音，极其霸道地刻在虚空中，震得整个大殿内的空气都在嗡嗡作响。

肖山感觉自己的膝盖微微一软，一股极其庞大、犹如实质山岳般的无形力量，极其粗暴地压在了他的肩头，试图将他的脊梁彻底压断，将他死死地按倒在地砖上。

他极其轻蔑地冷笑了一声，体内那股极其强悍、刚刚在雷劫中淬炼而成的筑基灵力瞬间开始疯狂流转。他硬是顶着那股足以压碎凡人骨骼的恐怖威压，极其缓慢、却又极其坚定地，站直了身体。

他低头看了一眼自己身上这套系统分配的官服——玄黑色的极其昂贵的底料上，用暗金色的丝线绣着代表着绝对公正的獬豸暗纹，腰间系着一条极其醒目的朱红色玉带，头上的乌纱帽上，插着一根代表着极高权力的白羽。

御史大夫。大乾帝国名义上的最高监察官，掌管整个帝国的司法刑狱，拥有纠察百官的极大权力，更有着皇帝亲赐的“代天巡狩”的极其恐怖的特权。

而在那条朱红色的玉带上，极其突兀地挂着一把不到三尺长的短剑。那把剑的剑鞘通体漆黑，没有任何华丽的纹饰，却散发着一股极其浓烈、令周围空气都近乎凝固的极致肃杀之气。

尚方宝剑。
拥有极其恐怖的系统特权：上可斩昏君，下可斩佞臣。

肖山瞬间明白了系统给他安排这个身份的底层逻辑。如果说赵无极是这个系统用来维持旧有僵化秩序的“核心基石”，那么他现在扮演的这个御史大夫身份，就是系统底层逻辑中，天生用来“破局”的那把极其锋利的利刃。

“肖山！太师当面问话，你竟敢狂妄不答？！”站在赵无极身后的一名赵派官员，极其嚣张地跳出来，指着肖山的鼻子厉声呵斥道。

肖山极其随意地拍了拍膝盖上并不存在的灰尘，那双犹如孤狼般冰冷的目光极其平静地扫过了满朝文武，最终，极其精准地落在了赵无极那张虚伪的脸上。

“赵太师，”肖山的声音虽然不大，但却带着一种极其奇异的穿透力，极其清晰地传遍了整个辽阔的金銮殿，“你一开口就问我该当何罪？我倒是非常想问问你——我奉了皇上的圣旨前往江南道巡查，在半路上极其蹊跷地遭遇了所谓的‘空间乱流’，九死一生才捡回一条命。我这前脚刚极其狼狈地踏入金殿，你后脚就这么迫不及待地想要给我定罪。你这到底是单纯地不想让我开口说话，还是你心里有鬼，怕我当着满朝文武的面，戳破你那些极其恶心的系统漏洞？”

满殿顿时一片哗然，犹如在一口沸腾的油锅里泼下了一瓢冷水。

在场的任何人都没有想到，这个向来在朝堂上极其低调、甚至有些懦弱的御史大夫，今天竟然像是吃了熊心豹子胆一样，敢在这代表着最高权力的金銮殿上，当面极其狂妄地顶撞权倾朝野的太师。

赵无极那双深邃的眼神微微眯了起来，眸子深处极其快速地闪过了一抹令人胆寒的冰冷杀意。但那抹杀意瞬间就被他用一张极其完美的政客面具彻底掩盖了过去。“看来江大人此番外出历练，真的是长了不少胆色。也罢，既然你能够平安归来，也是我大乾的一大幸事。本官今日就不去计较你这御前失仪的微末之罪了。来人，将江大人在江南道的监察卷宗呈上来吧。”

他试图用这种极其老辣、看似宽宏大量的官场手段，极其圆滑地化解掉肖山刚刚展露出的锐气，将话题重新拉回他能够完全掌控的节奏中。

但肖山，今天根本就不是来陪他玩这种极其无聊的政治过家家游戏的。

“太师莫急，这述职的事情，咱们可以稍后再说。”肖山极其突兀地向前跨出了一大步，他的目光极其无礼地越过了前面的赵无极，直接极其放肆地直视着那张高高在上的龙椅上，那道极其模糊的金色身影，“陛下，微臣此番前往江南道巡查，在极其偶然的情况下，发现了一件极其骇人听闻、足以动摇我大乾国本的天大丑闻！此事事关整个帝国的生死存亡，微臣不敢有丝毫隐瞒，必须立刻上报圣听！”

坐在龙椅上的那个极其年轻、甚至显得有些懦弱的皇帝，眼中极其罕见地闪过了一丝极其兴奋的光芒。他极其畏惧地看了一眼站在下方赵无极那宽阔的背影，然后用一种极力保持平稳、却依然透着一丝颤抖的语气说道：“爱卿……既然事关国本，那就但说无妨。”

“微臣今日，要在这金銮殿上，实名弹劾当朝太师赵无极——结党营私、极其贪婪地贪墨国帑、极其胆大包天地蒙蔽圣听、极其肆无忌惮地祸乱朝纲！其罪极其深重，足有十条之多，这十条大罪，条条都足以将其极其残忍地诛杀九族！”

肖山的声音，犹如一颗在金銮殿内被瞬间引爆的重磅炸弹，轰然炸开，震得大殿的穹顶都在瑟瑟发抖。

“放肆！极其放肆！”吏部侍郎孙正清像是一只被踩了尾巴的猫，极其激动地跳了出来，指着肖山的鼻子破口大骂，“肖山，你区区一介御史，安敢在这神圣的金銮殿上血口喷人，极其恶毒地污蔑国之栋梁的太师！”

“哦？污蔑？”肖山极其不屑地冷笑了一声，他极其从容地从宽大的袖中取出了一卷极其厚重的竹简，然后极其潇洒地扬手，将竹简在半空中哗啦啦地展开，“你说我污蔑？结党营私这一条，这里面有他赵无极与你孙正清侍郎，在这三年间极其隐秘往来的密信，整整十七封！贪墨国帑这一条，江南道这三年来的极其庞大的税银账目，每一笔极其隐蔽的坏账，其最终极其丑陋的流向，在这上面记录得清清楚楚！至于蒙蔽圣听，你们这些年极其嚣张地拦截的那些弹劾他的奏章，原件现在就极其完好地躺在中书省最深处的密阁里，只要陛下一道极其简单的圣旨，立刻就能将其公之于众！”

他每极其掷地有声地说出一句话，就极其坚定地向前迈出一步。

当他那句话说完的时候，他已经极其强势地走到了大殿的最中央，距离那位权倾朝野的赵无极，仅仅不过十步之遥。

赵无极脸上那种极其完美的从容，在这一刻，终于彻底消失了。

他极其冰冷、犹如看死人一般地盯着眼前的肖山，他身上那极其浓郁的紫色气运，开始极其狂暴、毫无规律地翻涌起来。

轰——！

一股极其恐怖、让人瞬间感到窒息的庞大威压，极其粗暴地从天而降。

那是属于渡劫期大能毫无保留的全力压迫，那是属于一品太师位阶的绝对压制，这两股极其恐怖的力量极其完美地叠加在一起，形成了一种近乎天道法则般的绝对碾压。在场的百官瞬间脸色剧变，那些修为极其低微的官员，甚至连一声惨叫都没发出来，就直接被这股恐怖的威压极其屈辱地压得跪倒在地，浑身不受控制地剧烈颤抖。

赵无极极其缓慢地伸出了一只手，对着大殿上方的虚空，极其用力地一握。

整个极其庞大的金銮殿内，瞬间响起了一声极其苍茫、仿佛来自远古的龙吟。

大乾的国运真龙。那是汇聚了亿万百姓极其盲目的信仰、以及大乾帝国整整三百年极其庞大气运，最终凝聚而成的一条极其庞大、散发着无尽威严的金色巨龙。它极其恐怖地从那片星斗大阵中探出了极其巨大的头颅。它那极其庞大的身躯，犹如一道不可逾越的山脉，极其霸道地横贯在整个金銮殿的上空。它那双散发着极其刺目金光的龙目，瞬间照亮了整座庞大无比的皇城。

奉天承运，天命所归。

这是封建帝王维持其极其残酷统治的绝对根基，也是这个沙盒世界里极其罕见的最高级别力量之一。而赵无极这个极其贪婪的权臣，在长年累月极其隐秘地把持朝政的过程中，早已经通过极其卑劣的手段，将自己的命格与这庞大的国运极其紧密地绑定在了一起。能够极其自如地调动这只国运真龙，这就是这位权臣用来镇压一切反抗的终极杀招。

“肖山，”赵无极的声音，此刻听起来就像是那高高在上的天道，在极其无情地宣判着一个极其渺小的蝼蚁的死刑，“你，可知罪？”

那颗极其庞大的金色龙头，极其缓慢地悬停在了肖山的头顶上方。它极其粗重地喷吐着极其灼热的龙息，让肖山周围的空气，都在瞬间极其剧烈地燃烧了起来。

“给本官，跪下！”赵无极发出一声极其恐怖的断喝。

那股威压瞬间以一种极其恐怖的几何倍数暴增，极其坚硬的金砖地面，以肖山站立的位置为绝对中心，发出了一声令人牙酸的“咔嚓”声，然后轰然龟裂，无数极其细小的裂纹像蜘蛛网一样极其迅速地向四周蔓延。

肖山的双腿发出了极其痛苦的骨骼摩擦声，他死死地咬着牙，极其顽强地扛着那股重达千钧的恐怖重压。他的膝盖在极其剧烈地颤抖，但他却极其固执地、一点一点地，再次极其笔挺地挺直了自己那仿佛永远也不会弯曲的脊梁。
