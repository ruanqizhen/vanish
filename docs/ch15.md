# 第十五章：大乾金殿的锋芒

## 大梦初醒

虚空碎裂的刹那，肖山只感觉身体被无数股力量撕扯、解构、再重组。

那种疼痛超越了肉身——像是有人直接在他的灵魂深处刻下烙印，每一行记忆、每一个身份标识都在被强行覆盖重写。视野中闪过无数光影：大乾帝国的疆域图、文武百官的命格星盘、金銮殿上龙椅的千年威压……这些信息如同洪水般强制塞入他的识海。

他被扔进了一个新的世界。

砰！

失重感骤然消失，他从半空中坠落，双膝重重地砸在冰冷坚硬的纯金地砖上。

大殿内瞬间鸦雀无声。

肖山撑着手臂抬起头。入目的是一座巍峨到令人窒息的古代宫殿。百丈高的金柱上盘绕着活物般的五爪金龙浮雕，龙目中吞吐着淡金色的灵气光晕。殿顶不见尽头，只有翻滚的云海和星斗大阵在其中沉浮。两侧伫立的文武官员个个气息如渊，周身灵压交织成一张无形的巨网。

这是大乾帝国的金銮殿。

而此刻，他正跪趴在这座权力中枢的正中央。

肖山的手指下意识地摸向胸口——隔着陌生的古代官袍，那块无针怀表依然传来冰凉的触感。沈策不在了。沈策用自己的命撕开裂缝，把他送到了这里。

他深吸了一口气，松开怀表，手掌握住了腰间剑鞘的边缘。

他的眼神变了。从一个在雷劫中苦苦求生的现代外卖员，变成了一把出鞘的冷锋。

"大胆！"

一声暴喝在大殿上方炸开。

肖山循声望去，只见文官队列之首，站着一位身着紫金蟒袍的中年男人。那人面如冠玉，三缕长髯垂胸，双目之中却仿佛蕴藏着两座深不见底的寒潭。

此人周身环绕着肉眼可见的紫色官气，那是大乾帝国一品大员才能凝聚的位阶威压，浓郁到几乎化为实质，在他头顶形成一片紫金色的华盖。更恐怖的是，他的修为赫然已至渡劫期巅峰——只差最后一步便能飞升仙界的存在。

朝中能穿紫金蟒袍的只有一人：太师，赵无极。

这个名字在肖山脑海中浮现的瞬间，被强行塞入的记忆便如泉水般涌出：赵无极，大乾三朝元老，官居太师之位，门下弟子遍布朝野，号称"半朝赵家人"。此人表面上忠君爱国，实则早已将整个帝国的权力网络编织成了自己的私有物。

而在肖山那看破虚妄的视界中，这位赵太师的真实身份远不止权臣那么简单。

肖山微微眯起眼睛。他刚刚筑基成功的双眼，看穿了这片世界的表象。

赵无极的信息面板在他的视野中展开：

【赵无极】
【职位：太师（正一品）】
【修为：渡劫期·九重天】
【气运值：9,999,999,999 / 10,000,000,000】
【身份：世界线修正锚点】
【威胁等级：致命】
【备注：该单位承载本世界80%剧情权重，击杀后可破除部分因果锁定】

看清了。

这老家伙根本不是什么历史人物，他是这个世界里最大的"毒瘤"，是一个伪装成人类的怪物！

"御史大夫肖山，面圣不跪，御前失仪，该当何罪！"

赵无极的声音并不高亢，却自带一种法则般的压迫力。每一个字都像是用震天之音刻在大殿的虚空中，震得周围的空气都在嗡嗡作响。

肖山的膝盖微微一软，一股无形的力量如同大山般压在他的肩头，试图将他整个人按倒在地。

他冷笑一声，强悍的筑基灵力在体内流转，硬生生顶住了那股威压，缓缓站直了身体。

他低头看了看自己身上的官服——玄黑色的底料上绣着獬豸神兽的暗纹，腰间系着朱红玉带，头顶的乌纱帽上插着一根象征监察权的白羽。

御史大夫，大乾帝国最高监察官，掌司法刑狱、纠察百官，有"代天巡狩"之权。

而在那玉带之上，赫然挂着一柄不到三尺的短剑。剑鞘漆黑如墨，没有任何纹饰，却散发着一种令整个大殿空气都为之凝固的肃杀之气。

尚方宝剑。
上斩昏君，下斩佞臣。

肖山明白了。如果说赵无极是这个世界里为了维持旧秩序而存在的"基石"，那么他现在被分配的这个御史大夫身份，就是一把天生的"破局"利刃。

"肖山！太师问话，你竟敢不答？！"旁边一名赵派官员厉声呵斥。

肖山拍了拍膝盖上的灰尘，目光平静地扫过满朝文武，最后落在赵无极的脸上。

"赵太师，"肖山的声音不大，却清晰地传遍了整座金銮殿，"你问我该当何罪？我倒想问问你——我奉旨巡查江南，半路遭遇'空间乱流'险些丧命，这才刚踏入金殿，你就急着给我定罪。是不想让我开口说话，还是怕我说出什么不该说的漏洞？"

此言一出，满殿哗然。

谁都没想到，这位平日里在朝堂上向来低调的御史大夫，今日竟敢如此当面顶撞权倾朝野的太师。

赵无极的眼神微微一眯，那双寒潭般的眸子里闪过一丝冰冷的杀意，但很快被他用完美的政客面具掩盖。"看来江大人此番外出，倒是长了不小的胆色。也罢，既然你平安归来，本官便不计较你的失仪之罪。将江南道的监察卷宗呈上来吧。"

他试图用居高临下的官场手段，把肖山的锐气化解于无形。

但肖山不是来陪他玩政治过家家的。

"述职的事不急，"肖山向前跨了一步，目光越过赵无极，直视龙椅上那道金色的身影，"陛下，臣此番巡查，发现了一件骇人听闻的大事，事关大乾国本的根本，臣不敢不报！"

龙椅上坐着的年轻皇帝眼中闪过一丝光芒，但随即又畏惧地看了一眼赵无极的背影，用尽量平稳的语气说："爱卿但说无妨。"

"臣要弹劾太师赵无极——结党营私、贪墨国帑、蒙蔽圣听、祸乱朝纲！其罪有十，条条当诛！"

肖山的声音如同一声炸雷，在金銮殿上轰然炸开。

"放肆！"赵无极的门生、吏部侍郎孙正清跳了出来，指着肖山的鼻子骂道，"你不过一介御史，安敢血口喷人，污蔑当朝太师！"

"哦？"肖山从袖中取出一卷厚厚的竹简（这同样是身份自带的道具），扬手展开，"我说他结党营私，有他与你孙侍郎往来的密信十七封为证；我说他贪墨国帑，江南道三年来的税银账目皆在此处，每一笔坏账的流向都清清楚楚；我说他蒙蔽圣听，这些年来所有弹劾他的奏章都被他在半路拦截了，原件还在中书省的密阁之中！"

他每说一句，便向前迈一步。

走到大殿中央时，距离赵无极已经不过十步之遥。

赵无极脸上的从容终于消失了。

他冷冷地盯着肖山，紫色的官气在他的周身狂暴翻涌。

轰——！

一股恐怖到令人窒息的威压从天而降。

那是渡劫期大能的全力压迫，更是大乾一品太师的位阶压制，二者叠加，形成了一种天地法则般的碾压。两侧的文武百官纷纷色变，修为稍低者直接被压得跪倒在地，浑身颤抖。

赵无极伸手在虚空中一握。

大殿之中响起了一声苍茫的龙吟。

那是大乾帝国的国运真龙。一条由亿万民众的信仰之力、三百年的王朝气运凝聚而成的金色巨龙，从大殿上空的星斗大阵中缓缓探出头来。它的身躯横贯整个金銮殿，龙目睁开时，金色的光芒照亮了整座皇城。

奉天承运。

这四个字，是帝王统治的根基，也是这个世界最高阶的力量。赵无极把持朝政多年，早已暗中将自己的命格与大乾国运强行绑定。调动国运真龙，便是他身为权臣的终极杀招。

"肖山，"赵无极的声音如同天道在宣判，"你可知罪？"

金色巨龙的庞大龙头悬停在肖山的头顶，龙息喷吐间，周围的空气都在剧烈燃烧。

"跪下！"赵无极一声断喝。

恐怖的威压猛然加重，金砖地面以肖山为中心轰然龟裂。

肖山的双腿发出一声令人牙酸的骨骼摩擦声，但他咬着牙，硬是扛着千钧重压，再次挺直了脊梁。

"赵太师，"肖山一字一句地开口，眼中闪过耀眼的精光，"你调动国运真龙来压我，靠的是你与我大乾国运绑定的那一丝因果，是也不是？"

赵无极眉头微皱，没有回答。

"你不说话，我就当你是默认了。"肖山冷笑一声，目光如炬，"那我问你——你与国运绑定的因果链，是建立在你三朝为官所积累的'合法'功勋和俸禄上，是也不是？"

"是又如何？"赵无极冷哼，"本官三朝元老，功在社稷，这份因果自然是天道认可的正统之链。"

"好一个天道认可的正统。"肖山举起手中的竹简，朗声道，"你的年俸是禄米三千石、银钱一万两。这四十五年来，你合法的总收入不过白银五十万两。"

肖山的声音越发洪亮，在大殿中回荡："但你名下的田产、商铺，估值白银三千万两。你门生进献的灰色收入，每年超过两千万两！赵太师，请你当堂解释一下，在没有逆天改命、没有点石成金的情况下，你的财富是如何实现连续四十五年、成百上千倍增长的？"

赵无极脸色铁青："一派胡言！那是本官祖上传下的家业！"

"错！"肖山猛地打断他，"你的发家史，本质上就是一个天大的笑话。这个谎言在情理上不成立，在天理上更不成立！"

肖山指向头顶的国运真龙：

"你的因果链，建立在一个虚假的、非法的罪孽之上！换言之，你用来蒙蔽天机、调动国运真龙的那个权柄，是伪造的！"

轰——！

国运真龙发出一声震天的咆哮。

那金色的龙目中，第一次出现了犹豫和纠结。它虽然是国运的凝聚体，但它遵循的是天地最根本的规则。当肖山的"天理质问"和赵无极的"虚假权柄"产生绝对冲突时，真龙陷入了混乱。

赵无极脸色剧变。他感觉到自己与国运真龙之间的连接正在断开。

"妖言惑众！"赵无极暴喝一声，浑身紫色官气暴涨，试图强行接管真龙。

"还不死心？"肖山向前跨了一步，"那我就彻底击溃你的虚伪面具！气运不是凭空捏造的，它需要功德来支撑。"

赵无极一愣。

肖山的指尖划过虚空，仿佛在拨动因果的丝线："你的因果链从一开始就沾满了业障。随着岁月的推移，这些罪孽会被不断放大。你的功德，实际上早就跌入了万劫不复的深渊。按照天道清算，你的气运正在以极快的速度雪崩！"

满朝文武都看呆了。他们虽然听不懂因果算计的细节，但他们能清晰地看到——随着肖山的话音落下，赵无极身上的紫色官气正在以肉眼可见的速度疯狂流失！

"住手！"赵无极彻底慌了，猛然抬手，一记蕴含着渡劫期全力的紫金色掌印朝着肖山轰然拍下。

如果这一掌拍实了，肖山绝对会当场灰飞烟灭。

就在那紫金掌印即将击中他的瞬间。

吼——！

国运真龙动了。它庞大的身躯猛然俯冲而下，横亘在肖山身前，用坚不可摧的龙鳞硬生生挡住了那一掌。

轰隆！

掌印在龙躯上爆碎，金銮殿剧烈震动。但国运真龙毫发无伤，而赵无极却被反震之力震得口吐鲜血，倒退了十多步。

"这、这怎么可能……"赵无极难以置信地看着自己的双手。

"还不明白吗？"肖山的声音从龙躯后方传来，带着绝对的冷酷，"你的虚假权柄已经被戳破了。国运真龙不再认可你，它现在听命于真正的天理。"

肖山缓缓拔出了腰间的尚方宝剑。

剑锋出鞘的刹那，大殿中所有的光芒都仿佛被吸入了那柄黑色的短剑中。

这把剑上没有附着一丝一毫的灵气，它所蕴含的力量比灵气更高一个维度——它承载的是天道赋予的"绝对裁决权"。在它的判定之下，没有修为的差别，唯有——合法与非法。

赵无极盯着那把剑，眼中终于浮现出深入骨髓的恐惧。他能看到，那把剑上缠绕着无数毁灭的法则。

"不、你不能杀我！"赵无极的声音变得尖锐、失真，带着明显的颤音，"我是三朝元老！我是大乾太师！我是国之栋梁！"

"你只是一个蛀虫。"

肖山举起尚方宝剑，剑锋直指赵无极。

"我以大乾御史大夫之名，裁定你——贪墨国帑、祸乱朝纲、乱臣贼子——就地正法！"

话音落下，肖山化作一道残影，筑基期的灵力全面爆发，直扑赵无极而去。

尚方宝剑的剑身上，骤然浮现出无数道银白色的天雷。

赵无极发出了绝望的嘶吼，祭出一方刻着"奉天承运"的镇国玉玺试图抵挡。但在尚方宝剑的天雷面前，那方伪造的玉玺如同纸糊的一般，瞬间被瓦解、粉碎、消散成金色的光点。

剑锋没有丝毫停顿，斩落在赵无极的脖颈上。

没有鲜血飞溅。

剑锋过处，赵无极的身体就像是风化的沙雕，从接触剑锋的那一点开始，迅速分解成无数细小的黑色飞灰。

"不……没有我……大乾会崩溃的……"

伴随着最后一声绝望的嘶喊，这位权倾朝野的三朝元老、渡劫期大能，彻底从这个天地间被抹杀了。

当光芒散去。

金銮殿的地面上，只剩下了一段长达数米的、像黑色死蛇一样不断扭曲的怪物残骸。它身上漂浮着无数诡异的符文，散发着刺鼻的焦糊味。

满朝文武噤若寒蝉，双腿发软地跪伏在地。

肖山收剑入鞘，冷冷地看着那段黑色残骸。

他的视野左上角，毫无预兆地弹出一行猩红色的系统提示：

**【检测到关键剧情节点已解锁】**
**【叙事引擎纠错补丁·第一锚点已清除】**
**【世界线自由变量已解锁至 14.2%】**
**【获得任务物品：锚点残卷（1/7）】**

紧接着，猩红色的警告字体开始疯狂闪烁：

**【警告：检测到锚点异常！】**
**【剩余残卷正在自行激活——数量：6】**
**【警告：锚点完全激活后将自动合并为'终极灭世协议'，届时世界将被强制重启。】**
**【剩余时间：未知。】**

肖山盯着那行猩红的字，握紧了尚方宝剑的剑柄。

他抬起头，目光穿透了金銮殿厚重的屋顶，看向大乾帝国那片虚假的、被编织的天空。

"还剩六个是吧。"

肖山冷笑一声。

"那就一个个，全他妈给老子斩干净。"
