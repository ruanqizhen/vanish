# 第十五章：大乾金殿的锋芒

## 大梦初醒

虚空碎裂刹那，肖山感觉身体被无数力量撕扯、解构、重组。

疼痛超越肉身——像有人在灵魂深处刻烙印，记忆、身份标识被强行覆盖重写。视野闪过光影：大乾疆域图、百官命格星盘、金銮殿千年威压……信息如洪水强制塞入识海。

被扔进了新世界。

砰！

失重感骤消，从半空坠落，双膝重重砸在冰冷纯金地砖上。

大殿鸦雀无声。

撑着手臂抬头。入目是巍峨的古代宫殿。百丈金柱盘绕活物般的五爪金龙，龙目吞吐淡金灵气。殿顶不见尽头，只有翻滚云海和星斗大阵沉浮。两侧文武官员气息如渊，灵压交织成无形巨网。

大乾帝国金銮殿。

此刻正跪趴在权力中枢正中央。

手指下意识摸胸口——隔着官袍，无针怀表传来冰凉。沈策不在了。沈策用命撕开裂缝，把他送到了这。

深吸气，松开怀表，手掌握住腰间剑鞘。

眼神变了。从雷劫中求生的外卖员，变成出鞘冷锋。

"大胆！"

一声暴喝在大殿炸开。

循声望去，文官之首站着身着紫金蟒袍的中年男人。面如冠玉，长髯垂胸，双目如深不见底的寒潭。

周身环绕紫气，一品大员的位阶威压，浓郁得几乎化为实质，在头顶形成紫金华盖。更恐怖的是，修为已至渡劫巅峰——差一步飞升。

朝中穿紫金蟒袍的只有一人：太师，赵无极。

名字浮现瞬间，记忆如泉涌：三朝元老，门下弟子遍布朝野，号"半朝赵家"。表面忠君，实则将帝国权力网编织成私有物。

而在肖山视界中，这位太师真实身份远不止权臣。

微微眯眼。刚筑基成功的双眼，看穿世界表象。

赵无极信息面板在视野展开：

【赵无极】
【职位：太师（正一品）】
【修为：渡劫期·九重天】
【气运值：9,999,999,999 / 10,000,000,000】
【身份：世界线修正锚点】
【威胁等级：致命】
【备注：承载本世界80%剧情权重，击杀可破除部分因果锁定】

看清了。

这老家伙根本不是历史人物，是这世界最大"毒瘤"，伪装成人类的怪物！

"御史大夫肖山，面圣不跪，御前失仪，该当何罪！"

赵无极声音不高，却带法则般压迫力。每个字像震天音刻在虚空，震得空气嗡嗡作响。

肖山膝盖微软，无形力量如山压在肩头，试图将他按倒。

冷笑一声，强悍筑基灵力流转，硬顶住威压，缓缓站直。

低头看身上的官服——玄黑底料绣獬豸暗纹，腰系朱红玉带，乌纱帽插白羽。

御史大夫，最高监察官，掌司法刑狱、纠察百官，有"代天巡狩"权。

玉带上，挂着不到三尺的短剑。剑鞘漆黑，没任何纹饰，却散发令空气凝固的肃杀气。

尚方宝剑。
上斩昏君，下斩佞臣。

明白了。如果赵无极是维持旧秩序的"基石"，他这御史大夫身份，就是天生"破局"利刃。

"肖山！太师问话，竟敢不答？！"赵派官员厉声呵斥。

肖山拍拍膝盖灰尘，目光平静扫过满朝文武，落在赵无极脸上。

"赵太师，"肖山声音不大，却传遍金銮殿，"问我该当何罪？我倒想问你——我奉旨巡查江南，半路遭遇'空间乱流'险些丧命，刚踏入金殿，你就急着定罪。不想让我开口，还是怕我说出漏洞？"

满殿哗然。

谁都没想到，向来低调的御史大夫，今日敢当面顶撞权倾朝野的太师。

赵无极眼神微眯，眸子闪过冰冷杀意，随即用政客面具掩盖。"看来江大人此番外出，长了胆色。也罢，既然平安归来，本官不计较失仪之罪。将江南道监察卷宗呈上来。"

试图用官场手段化解锐气。

但肖山不是来陪他玩政治的。

"述职不急，"向前跨一步，目光越过赵无极，直视龙椅上的金色身影，"陛下，臣此番巡查，发现骇人听闻的大事，事关国本，不敢不报！"

年轻皇帝眼中闪过光芒，畏惧地看了一眼赵无极背影，平稳语气说："爱卿但说无妨。"

"臣弹劾太师赵无极——结党营私、贪墨国帑、蒙蔽圣听、祸乱朝纲！其罪有十，条条当诛！"

声音如炸雷，在金銮殿轰然炸开。

"放肆！"吏部侍郎孙正清跳出指着肖山鼻子，"你一介御史，安敢血口喷人，污蔑太师！"

"哦？"肖山从袖中取出竹简，扬手展开，"结党营私，有与你孙侍郎往来密信十七封；贪墨国帑，江南道三年税银账目在此，每笔坏账流向清清楚楚；蒙蔽圣听，这些年弹劾他的奏章全在半路拦截，原件在中书省密阁！"

每说一句，向前迈一步。

走到大殿中央，距赵无极不过十步。

赵无极从容消失。

冷冷盯着肖山，紫气狂暴翻涌。

轰——！

恐怖窒息的威压从天而降。

渡劫大能全力压迫，一品太师位阶压制，叠加成法则碾压。百官色变，修为低者直接跪倒颤抖。

赵无极伸手虚空一握。

大殿响起苍茫龙吟。

国运真龙。亿万信仰、三百年气运凝聚的金色巨龙，从星斗大阵探出头。身躯横贯金銮殿，龙目光芒照亮皇城。

奉天承运。

帝王统治根基，这世界最高力量。赵无极把持朝政，早将命格与国运绑定。调动真龙，是权臣终极杀招。

"肖山，"赵无极声音如天道宣判，"可知罪？"

庞大龙头悬停肖山头顶，龙息喷吐，空气剧烈燃烧。

"跪下！"一声断喝。

威压猛重，金砖地面以肖山为中心轰然龟裂。

双腿发出骨骼摩擦声，咬着牙扛着千钧重压，再次挺直脊梁。

"赵太师，"肖山一字一句开口，眼中闪过精光，"调动国运真龙压我，靠你与国运绑定的因果，是也不是？"

赵无极眉头微皱，没回答。

"不说话，当默认了。"肖山冷笑，"我问你——你与国运绑定的因果链，建立在你三朝为官积累的'合法'功勋和俸禄上，是也不是？"

"是又如何？"冷哼，"本官三朝元老，功在社稷，这份因果自然是天道认可的正统之链。"

"好一个正统。"肖山举起竹简，"你年俸禄米三千石、银一万两。四十五年，合法总收入不过白银五十万两。"

声音洪亮在殿中回荡："但你名下田产商铺估值三千万两。灰色收入每年超两千万两！请解释，在没逆天改命、点石成金情况下，财富如何连续四十五年成百上千倍增长？"

赵无极脸色铁青："一派胡言！那是祖上家业！"

"错！"猛地打断，"你的发家史是天大笑话。这谎言在情理不成立，在天理更不成立！"

指向头顶国运真龙：

"你的因果链，建立在虚假、非法罪孽之上！用来蒙蔽天机、调动真龙的权柄，是伪造的！"

轰——！

国运真龙发出震天咆哮。

龙目中第一次出现犹豫。它遵循天地根本规则。当肖山"天理质问"和赵无极"虚假权柄"产生绝对冲突，真龙陷入混乱。

赵无极脸色剧变。感觉到与真龙连接正在断开。

"妖言惑众！"暴喝，紫气暴涨，试图强行接管真龙。

"还不死心？"肖山向前跨步，"那就彻底击溃你！气运不是凭空捏造，需要功德支撑。"

赵无极一愣。

指尖划过虚空，仿佛拨动因果丝线："因果链从一开始沾满业障。岁月推移罪孽放大。你的功德早跌入深渊。按天道清算，气运正在雪崩！"

满朝文武看呆。虽听不懂细节，但能看到——随着肖山话音，赵无极身上的紫色官气正肉眼可见地疯狂流失！

"住手！"彻底慌了，猛抬手，一记蕴含渡劫全力的紫金掌印朝肖山轰然拍下。

拍实了，绝对灰飞烟灭。

就在掌印即将击中瞬间。

吼——！

国运真龙动了。庞大身躯俯冲，横亘身前，用龙鳞硬生生挡住那一掌。

轰隆！

掌印在龙躯爆碎，金銮殿震动。真龙毫发无伤，赵无极被反震吐血，倒退十多步。

"这、怎么可能……"难以置信看着双手。

"还不明白？"肖山声音从龙躯后传来，绝对冷酷，"虚假权柄被戳破了。国运真龙不再认可你，它听命于真正的天理。"

缓缓拔出尚方宝剑。

剑锋出鞘刹那，大殿光芒仿佛被吸入黑色短剑。

剑上没附着灵气，蕴含力量比灵气高一个维度——承载天道赋予的"绝对裁决权"。判定下没修为差别，唯有——合法与非法。

赵无极盯着剑，浮现深入骨髓恐惧。能看到剑上缠绕无数毁灭法则。

"不、你不能杀我！"声音尖锐失真，"我是三朝元老！大乾太师！国之栋梁！"

"你只是个蛀虫。"

举起宝剑，剑锋直指。

"我以大乾御史大夫之名，裁定你——贪墨国帑、祸乱朝纲——就地正法！"

话音落下，化作残影，筑基灵力全面爆发直扑而去。

剑身上骤然浮现无数银三天雷。

赵无极绝望嘶吼，祭出刻着"奉天承运"镇国玉玺抵挡。但在天雷前，伪造玉玺如同纸糊瞬间瓦解消散。

剑锋没丝毫停顿，斩落脖颈。

没有鲜血飞溅。

剑锋过处，身体像风化沙雕，迅速分解成无数黑色飞灰。

"不……没有我……大乾会崩溃的……"

伴随最后嘶喊，权倾朝野的三朝元老彻底从天地间被抹杀。

光芒散去。

地面只剩下一段长数米、像黑色死蛇扭曲的怪物残骸。漂浮诡异符文，散发焦糊味。

满朝文武噤若寒蝉，双腿发软跪伏。

肖山收剑，冷冷看着残骸。

视野左上角弹出猩红系统提示：

**【检测到关键剧情节点已解锁】**
**【叙事引擎纠错补丁·第一锚点已清除】**
**【世界线自由变量已解锁至 14.2%】**
**【获得任务物品：锚点残卷（1/7）】**

紧接着，警告字体疯狂闪烁：

**【警告：检测到锚点异常！】**
**【剩余残卷正在自行激活——数量：6】**
**【警告：锚点完全激活后将合并为'终极灭世协议'，届时世界强制重启。】**
**【剩余时间：未知。】**

盯着那行猩红的字，握紧剑柄。

抬起头，目光穿透厚重屋顶，看向那片虚假的天空。

"还剩六个是吧。"

冷笑。

"那就一个个，全他妈给老子斩干净。"
