# 第二十八章：阳光下的外卖员与总裁

林江市的时钟拨到了六月，阳光明媚。

阳光从梧桐叶的缝隙中倾泻，在柏油路面上投下一地斑驳的阴影。新城区的街道上车流如织，汽车的鸣笛声混杂着环境的白噪音，构成了一幅再普通不过的平凡日常画面。

## 褪色的外卖员

肖山跨着那辆灰色的电动车，眯眼看着前方拥堵的车流，嘴里叼着一根补充体力的糖。

他身上穿着那件洗得发白的外卖员制服，黄色的衣服在无数次洗涤后已经褪色，但洗得很干净，散发着淡淡的肥皂气味。后背上外卖平台的Logo已经被磨得模糊，却依然能看出那只奔跑的小袋鼠。

这身衣服，他已经穿了快两个月了。

自从放弃了高维控制、退出那场生死的漩涡之后，他就重新注册了外卖平台的普通账号。不是为了体验生活，也不是为了反转，他只是单纯想做一些最简单的体力工作，让自己从紧绷的状态中释放出来。

送外卖这件事，就是最简单的任务。

取餐，送餐，看到顾客满意的表情，然后穿行在这座城市里。这种规律而平静的生活，反而给了他从所未有的踏实感。

他甚至觉得，如果日子就这么过下去，也挺好。

——如果后座上没有坐着那个引人注目的家伙的话。

"山哥，你糖能分我一半吗？"

身后传来低沉慵懒的声音，嗓音里含着笑意。

后背传来温热的体温，腰间那双手臂收得更紧。

他翻了个白眼。

"沈策，你他妈能不能坐好？你这样我快被你晃倒了。"

"我抱得很稳啊。"沈策的声音无辜，"我抓得很紧，绝对不会掉下去。"

"你是不会掉下去，我快被你勒死了！"

肖山没好气地拍了拍腰间的手臂，但那双臂像铁箍一样，纹丝不动。

他无奈叹气，从嘴里拿出糖，往身后一递："最后一颗，草莓味的。"

身后的男人沉默一秒，低笑。

接着，肖山感觉指尖一空——糖被叼走了。

不是用手。是用嘴。

温热柔软的触感在指尖一触即离，肖山的耳朵瞬间红透了。

"沈！策！"

"嗯？"沈策含着戏谑，"怎么了山哥？这糖挺甜的，就是有点化了。"

"你——你他妈不用手接？！"

"我腾不出手啊。"沈策理直气壮，"再说了，你都送到嘴边了，我要是用手，不是浪费时间吗？"

肖山深吸一口气，告诉自己不要跟这个混蛋生气。

这两个月，他已经习惯了沈策这种无孔不入的调戏，但每次还是会心跳加速。

更让他无语的是——

## 高定西装

沈策穿着一身高定西装。

是的，极其奢华的高级定制西装。

意大利手工面料，深灰色带暗纹，剪裁利落到每一寸线条都完美贴合他的身材。袖口的金属袖扣在阳光下泛着高级的光泽。

这个人配上这身衣服，别说坐破电动车了，就是站在垃圾回收站旁边，也能拍出时尚大片。

这两个人形成的画风，太过诡异——

一个穿着褪色外卖服的外卖员，骑着破电动车；
一个穿戴着高定西装、帅得惊天动地的总裁，坐在他的后座上，搂着腰，嘴里叼着糖。

任何路人看到这幅画面，第一反应都是：自己的眼睛是不是出了问题？

事实上，从他们发车到现在，十五分钟里，肖山体验了前所未有的注目礼。

## 衣着反差引发的围观

首先是路边等公交的大妈。

她原本在看手机，余光扫到电动车过去，下意识抬头，然后惊得合不拢嘴。她扯着旁边的朋友："那个……后面那个是明星吧？肯定是！"

"前面那个骑车的是助理？哪家助理穿外卖服？"

肖山听着身后的议论，嘴角抽搐。

然后是开宝马的西装男。

他等红灯时百无聊赖，看到沈策时整个人极度震惊。车窗降下，死死盯着沈策的西装。

"卧槽？Brioni 的限量版？那个人穿的是 Brioni！"西装男的声音充满崩溃，"Brioni 的定制服，坐在破电动车上？！叼着糖？！这个世界疯了吗？！"

肖山差点没忍住笑出声。

他回头对沈策压低声音："听见没？人家认出你这身西装了。Brioni 的定制套装，坐我这破车上，你不觉得掉价？"

沈策挑眉，把糖拿出来晃了晃："掉价？我不觉得啊。"

他把脸贴在肖山后背，声音慵懒："这衣服再贵，也就是件衣服。能陪你送外卖，是它这辈子的荣幸。"

肖山呼吸一滞。这人说话怎么直击人心。

"少来这套。"肖山强行镇定，"把你送到目的地后，赶紧换个普通衣服，别到时候被路人拍照发到网上，又要上热搜。"

"发就发呗。"沈策满不在乎，"让公司的公关部门找点事做，省得天天闲置。"

肖山："……"你家公关部是这么用的？

电动车继续前行，驶入了一条繁华的商业街。两旁全是奢侈品店，路人都是光鲜亮丽的都市精英。

在这样的背景下，肖山的外卖服显得格格不入。但更扎眼的是后座的沈策。

两个白领女性看到沈策，停下了脚步。"你看那个！好帅的人！"
"西装看起来好贵，怎么坐外卖车啊？"
"管他怎么坐，颜值逆天！快拍照发朋友圈！"

肖山听到身后的"咔嚓"声，头皮发麻："沈策，她们在拍我们。"

"嗯，拍吧。"沈策云淡风轻，"我今天的状态不错。"

"我是不想明天上同城热搜！"肖山咬牙，"现在短视频多离谱你不知道？要是被人认出你的真实身份，咱俩就真被人肉围观了。"

"那不是挺好？"沈策笑，"我买点热搜，把你推成网红，你接单接得手软，比我开公司赚钱快。"

肖山翻白眼。他搞不懂沈策的逻辑。身家几十个亿的大老板，穿着高定坐在外卖车上笑得那么开心。要是换以前的"沈三爷"，这种环境他三分钟就会发飙。

但现在，他不但坐了，还觉得很舒服。

电动车在红灯前停下。肖山单脚撑地，回头看沈策。阳光在他脸上勾勒出完美的阴影。

这个男人是真的好看。好看到即使坐在破车上，也像在拍电影。

肖山收回目光，暗骂自己——都看了两个月了，怎么还是没有免疫这个家伙。

## 路口偶遇

身后几米，一辆黑色保时捷停下。车窗降下，一个戴墨镜的人目光锁定了沈策。

"沈……沈总？"

那人四十来岁，商务休闲装。他盯着沈策确认没有认错人后，表情极其精彩。

"你是沈氏的沈总吧？"

沈策懒洋洋看他："你是？"

"我姓陈，鼎丰资本的，上次酒会见过。"陈总目光在沈策和电动车间扫视，"沈总这是……体验生活？"

沈策笑了一声："不是。"他手臂紧了紧，下巴搁在肖山肩上，"我陪我男朋友送外卖。"

空气安静了三秒。

陈总的嘴张成 O 型，眼神来回打量了七八次，最后停留在沈策的手上，艰难咽口水。

"男……男朋友？"

"嗯，伴侣。"沈策笑得云淡风轻，"很帅吧？"

肖山想找个地缝钻进去。

陈总被这消息震惊得呆住了，看着肖山洗白的外卖服和沈策的定制西装，不知道该怎么接话。

绿灯亮了。

"陈总，我们先送下一单了。"沈策扬下巴，"有空聊聊。"

拍了拍肖山："走吧山哥。"

肖山如蒙大赦，拧动把手，电动车"嗡"地窜了出去。风吹散了尴尬，沈策在身后笑出声。

"你还笑！"肖山没好气，"明天圈子里怎么传你？'沈氏总裁坐在外卖车上'？这消息太有意思了？"

"我为什么不高兴？"沈策声音温柔，靠在他肩上，"我就是在向所有人宣布，我现在过得非常好。"

肖山沉默，手指收紧。

"再说了，"沈策语气欠揍，"能让我穿 Brioni 坐在后座的人，全世界也就你一个。这不值得高兴？"

肖山轻哼，嘴角却不受控制地上扬。

## 遗留的青瓷杯

电动车沿着滨江路前行，这是肖山的日常路线。他对每一个路口、每一个红绿灯、每一家店的位置了如指掌。

路边店面向后掠去。这种简单的生活很普通，但比以前踏实。不用提防暗处的敌人，不用在复杂的阴谋里周旋，只需一单接一单地送。家里还有沈策在等。

"山哥，下一单去哪？"

肖山看手机屏幕："江滨南路 188 号，鸿远大厦 12 楼，台式卤肉饭和波霸奶茶。"

"那家卤肉饭的味道怎么样？"沈策笑。

"还行。比上个月新的强。"

沈策笑出声。肖山通过后视镜看他，阳光照在他脸上，让人恍惚。

想起初见，那个浑身是血的沈策，像个濒临崩溃的人。现在，却成了最重要的伴侣。

电动车拐过江东路，肖山减速。

站点的陈设更新了。以前的破旧海报不见了，换成了崭新的铝合金玻璃门。窗台的枯死绿萝换成了两盆新鲜的吊兰。里面坐着个普通的老大爷，不是以前那个阴鸷的眼线了。

一切都重置了。

但窗台上还放着一个遗留物。

一只青瓷茶杯。

它安静地坐在角落，泛着温润的光泽。杯口有细小的裂痕——不深，但像是一条无法抹平的伤疤。

肖山认识这只杯子。是昔瑶的。

他停车，走过去拿起。翻过杯底——刻着极小的字，像是在底部直接刻上去的：

`"0号档案——已存。董昔瑶留。"`

肖山愣住。0号档案。不是一号。是零号。

沈策从后座探头，目光落在字迹上，停留了很久。

然后他伸手，从肖山手中接过杯子。动作很轻，带着谨慎，像怕弄碎易碎的瓷器。他沉默着，眼睑低垂，然后把杯子收进风衣内袋——贴近胸口的位置，妥帖保存。

肖山注意到他指尖在发抖，上车后，眼眶有些发红。

但肖山没问。重新拧动把手，驶离江东路口。

## 私密餐厅（Taipei Gourmet）

车子驶入林荫道，车流变少。

"山哥，想过以后的人生怎么规划吗？"沈策问。

肖山想过。以前目标是活下去、赚够钱、找个没人的角落待着。现在，他想和沈策在同一个世界里。

"你呢？"肖山反问。

"我想和你永远在一起。"沈策说得轻描淡写，手臂却收紧了，"公司的事我交给了信托公司和职业经理人打理。以后我会轻松，不用天天处理繁重的工作。"

"舍得放弃那么多权力？"

"钱没了可以再赚。你要是没了——那我才是真毁了。"沈策声音温柔。

肖山鼻子发酸，猛地加速："少说肉麻话，大总裁说这个不嫌丢人。"

"丢什么人？夸我老婆，天经地义。"

电动车穿过林荫道，重新汇入主路。路况拥堵，前面的十字路口发生了拥堵，车流排长队。

旁边一辆奔驰车窗降下，格子衫年轻人探头，夸张地看着沈策："兄弟，你这西装挺贵啊，怎么坐外卖车上？不搭啊。"

周围车里的人都笑了。肖山脸发烫。

沈策毫不在意，拍拍肖山："我这坐骑怎么了？这个人全世界唯一，比你的奔驰贵多了。"

年轻人被噎住，周围笑声多了几分善意。

电动车挤过拥堵，重新加速。

到了目的地——"Taipei Gourmet"餐厅。

招牌是木质底，这家店很受白领欢迎。肖山来过很多次。他减慢车速靠边。

就在这时，沈策没有下车，下巴搁在肖山肩上："山哥，你有没有查过这家店的老板是谁？"

肖山一愣，转头。余光扫到餐厅门口的一块黄铜牌子。

他看过去，瞳孔猛地收缩。

牌子上写着：
`"台北 Gourmet`
`——献给肖山。`
`(愿你往后余生，每一天都有人陪。)"`

落款：
`"Shen Ce (沈策)"`

肖山大脑瞬间空白。他看向沈策。沈策笑眯眯地看着他。

"你……"

"我买下了这家店。"沈策轻描淡写，"你不是喜欢这家的味道吗？我寻思既然喜欢，就把店买下来送你。以后想怎么改菜单就怎么改，想加什么料就加什么。可以专门做'肖山特供'。"

肖山说不出话。

"愿你往后余生，每一天都有人陪。"

难怪这两个月，沈策老拉他来这吃饭。难怪店员对他格外热情。原来沈策早就买下了这家店。

"你……花这么多钱，就为了让我吃口卤肉饭？"肖山声音发哑。

沈策笑，拉他过来在额头轻吻："不只是卤肉饭。这家店，是我们新生活开始的地方。"

"你重新送外卖的第一天，第一单就是这家店。送完后，你在这台阶上坐了二十分钟。看起来很孤单。我当时就在二楼看着你——我想，如果能让你以后不再露出那种背影，我愿意提供一切支持。"

肖山呼吸一滞。他想起来了。那天他很迷茫，不知道未来怎么办。然后沈策发来消息："回家吧，我在等你。"

"山哥，我不会放下你。"沈策握紧他的手，"不管你做什么，我都做你最坚实的后盾。你送外卖，我就坐你后座；你重新创业，我给你提供一切支持。"

肖山深吸气，忍住眼眶的热意，别过头："少吹牛了，先把西装换了。"

沈策笑，从车上跳下。

就在这时——肖山的手机"叮咚"一声。

是一条新的消息。

肖山低头，看清内容时，整个人愣住。

未知发件人。配送地址：

`"翠微山·白楼 (终点塔旧址)。"`

备注栏：
`"菜要热的。我们在等你。"`

肖山盯着那几个字，手指微颤。这种理所当然的、带点唠叨的语气。太熟悉了。

沈策也看到了消息，沉默几秒，轻笑："看来，有些沉睡的人，醒了。"

肖山把手机放回口袋，发动车。

"走不走？"

"走。"沈策坐上后座，"不过今天先不去。先把这顿饭吃完。看来有人还不甘心，那就去见见。"

阳光依然灿烂，但空气中有一扇未知的门被悄然打开了。

肖山看着远处的天际线，笑了。

"总有一天，我们会用自己的双脚，走到那个人面前。"

沈策搂紧了他："不管去哪，我都陪你。"

电动车迎着阳光，驶向远处的翠微山方向。

在下一声"叮咚"之前，故事，远未结束。

（第二十八章 完）
