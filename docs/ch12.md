# 第十二章：高维降临与金丹

## 废墟中的光芒

庞大而空旷的地下空间里，死寂得令人感到窒息。

肖山大口喘着粗气，站在那块已经四分五裂的青石碑前，手中依然死死紧攥着那半截象征着“木偶师”的红色人皮。他能清晰地感觉到自己心脏跳动的声音，在经历了刚才那种极致的生死一线后，他的心跳非但没有虚弱，反而正以一种极其沉缓、有力的节奏跳动着。

赢了。
他用一种最暴烈、最不计后果的因果悖论，强行摧毁了这个规则怪谈的底层抹杀规则。

“结束了吗？”肖山低声问道，沙哑的声音在空旷的地下室里回荡。

沈策没有立刻回答。他依然握着那把卷刃的银餐刀，站在几米外的地方，眉头紧锁，那双锐利的眼睛，死死盯着那个厉鬼崩溃消散的地方。

“不对劲。”沈策的声音里，透出了一种极其罕见的、如临大敌的凝重。

肖山心中一凛，顺着他的目光看去。

在刚才那个厉鬼轰然溃散的中心位置，并没有像普通的灵异怪谈那样，化为一堆毫无意义的飞灰。相反，在那里，有一团极其纯粹、极其耀眼的光芒，正在黑暗中缓慢地凝聚成型。

那是一个只有拳头大小的球体。
它散发着一种温润如玉，却又极其刺目的金色光芒。在那层金色的光晕之中，隐约可以看到无数极其细小、繁复到了极点的金色符文，正在不断地流转、生灭。与这片地下空间那种阴冷、压抑、死寂的气息截然不同，这颗金色的球体，散发着一种浩瀚、古老，仿佛蕴含着一个微型宇宙般磅礴的恐怖生机。

“那是个什么鬼东西？”肖山忍不住走近了两步，他甚至能感觉到一股极其温暖、甚至有些烫人的气流扑面而来。

“那个厉鬼在彻底崩溃前，已经在这个沙盒里，吞噬了几百个人的精气，以及过去十年间积攒下来的极其庞大的怨念。当它的底层规则被你强行撑爆的瞬间，那些庞大而驳杂的能量无处宣泄，在极致的空间压力下，发生了极其严重的向内坍缩。”

沈策缓缓走到那团金光面前，一向古井无波的眼神中，竟然闪过了一丝极其强烈的震惊。
“这东西……如果在那些极其古老、只存在于传说中的高维修真体系里，应该被称为——金丹。”

肖山彻底愣住了，大脑有一瞬间的短路。
金丹？修仙小说里那种吃一颗就能白日飞升、脱胎换骨的金丹？
他妈的一个靠敲键盘写代码的底层程序员，在一个赛博诡异风格的现代都市规则怪谈里，竟然硬生生地爆出了一颗修真界的金丹？！

这画风，错乱得简直离谱！

“它怎么会出现在这种地方？”肖山下意识地伸出手想要去触碰那团金光，却被沈策一把极其严厉地拦住了。

“别碰它！这里面的能量极其狂暴、混乱，你现在的凡人之躯，只要碰一下就会被炸得灰飞烟灭。”沈策沉声道，“这个世界的底层逻辑，比我们想象的还要混乱、还要无序。它不仅仅只是一套简单的代码法则，里面绝对还被强行混杂了其他更高维度的修炼体系法则。在系统彻底崩溃的时候，它只能把那些它自己都无法解析的残余能量，强行封装成了这种极其不稳定的物理形式。”

轰隆隆——

就在两人说话间，整个庞大的地下空间，突然毫无征兆地开始剧烈摇晃起来。
那绝对不是普通的地震，而是从这片空间的物理结构内部，传来的一种令人绝望的崩塌感。头顶那些坚固的青砖，开始化作极其细小的粉末，如同下雨一般簌簌落下。

肖山猛地抬起头，瞳孔瞬间紧缩到了极点。

透过那正在迅速崩解的天花板，他看到了外面的世界。
但那，已经不再是他所熟悉的林江市了。

## 高维抹杀

原本属于林江市的夜空，此刻已经变成了一种令人心悸的深沉紫黑色。没有一颗星光，没有一丝云层，只有绝对的死寂。
而在那无尽的虚空最深处，正缓缓降下一种完全无法用人类语言去描述的“光”。

那道光，没有任何的颜色，甚至超越了人类视网膜能够解析的极限。但它却带着一种绝对的、无可违逆的、令人灵魂战栗的抹杀意志。光芒所扫过的地方，无论是高耸的摩天大楼、宽阔的街道，甚至是奔流的江水，全都在一瞬间失去了三维空间的厚度，变成了极其扁平的画纸，然后在下一个瞬间，碎裂成了漫天毫无意义的虚无飞灰。

“那是……”肖山只觉得浑身的血液都在这一刻彻底倒流，一种发自灵魂深处的恐惧，死死地攫住了他的心脏。

“高维物理清洗。”沈策的声音冷到了骨髓里，“你用因果悖论摧毁了那个厉鬼，但也同时彻底打破了这个区域原本极其脆弱的规则平衡。这个世界背后的天地法则检测到了无法自我修复的严重异常，所以，它根本不打算修补了，它要直接把这片区域，连同里面所有的存在，彻底抹除。”

“抹除整个林江市？！”肖山难以置信地大吼。

“它的目标，只有我们。整个林江市，只不过是因为处于同一个坐标轴，而被无辜牵连的沙盘罢了。”沈策缓缓转过头，看着肖山。在他那双一向波澜不惊的眼睛里，肖山第一次，看到了一种令人心碎的、决绝的死意。

“没时间了。”

沈策突然毫无预兆地出手，一把抓住了半空中那颗极其狂暴的金丹。
金光触碰他皮肤的瞬间，立刻发出了一阵令人牙酸的“嗤嗤”灼烧声。沈策的手掌瞬间被烤得焦黑一片，皮开肉绽，但他竟然连眉头都没有皱一下。

他猛地转过身，用一种根本无法抗拒的恐怖力量，死死地捏住了肖山的下巴，然后，将那颗滚烫到了极点的金丹，极其粗暴地塞进了肖山的嘴里。

“给我咽下去！”

“唔——！”肖山瞪大了眼睛，拼命地想要挣扎，想要把那颗烫嘴的东西吐出来。

但那颗金丹入口即化，瞬间化作了一股极其狂暴、滚烫如岩浆般的金色洪流，直接冲入了他的腹中。剧烈的灼烧感在瞬间剥夺了他所有的物理感官，他的五脏六腑仿佛正在被地狱的烈火疯狂焚烧，他体内原本脆弱的经脉被那股野蛮的力量强行拓宽、撕裂，然后又在金光中被极其粗暴地重组！

“啊啊啊啊——！”肖山痛苦地跪倒在地上，发出了一声犹如野兽般的凄厉嘶吼，他的皮肤表面，开始向外渗出极其耀眼、刺目的金色光芒。

“听着！”沈策一把揪住肖山的衣领，将他从地上硬生生地提了起来，他的声音在周围空间的轰鸣崩塌声中，依然清晰无比，“这颗金丹里蕴含的能量，足以护住你的灵魂，让你在时空乱流中不至于被瞬间撕碎。这是你现在唯一活下去的机会！”

“那你呢？！”肖山双眼赤红，强忍着体内那种几乎要将他撕裂的剧痛，死死抓住了沈策那只已经焦黑的手臂。

沈策看着他，突然笑了。
那是肖山认识他这么久以来，第一次看到他笑。那个总是带着冰冷面具、杀伐果断的系统执行官，此刻的笑容里，竟然透着一种令人心酸的释然与温情。

“我只是这个局里的一个死人，肖山。”沈策轻声说道，“我存在的唯一使命，就是把你，送出去。”

毁灭的光束，已经彻底穿透了地下室那本就不堪一击的防御。
周围的一切都在分崩离析，坚硬的青砖化作了最细微的齑粉，空气被压缩到了令人窒息的极致。

沈策极其果断地松开了肖山，他从怀里，掏出了那块极其眼熟的东西。

那是一块银色的、表面没有任何指针和刻度、甚至布满了裂痕的古旧怀表。

他左手紧紧握住那块怀表，用那根已经被金丹灼烧得深可见骨的拇指，狠狠地按下了顶端的机括。

咔嚓。

怀表表面爆发出了一阵极其刺目、深邃如海的深蓝色强光。那道光芒带着一种极高维度的空间力量，在两人的面前，硬生生地撕开了一道巨大的空间裂缝。而在裂缝的另一端，不是正在崩溃的林江市，而是深邃、黑暗、狂暴无序的虚空乱流。

“走吧。别回头。”

沈策用尽了全身最后的一丝力气，在肖山的胸口，极其用力地猛推了一把。

肖山的身体完全不受控制地向后倒去，直直地跌入了那道深蓝色的空间裂缝之中。
在他视线即将被黑暗彻底吞没的最后一刻，他看到了令他肝胆俱裂的最后一幕——

高维的毁灭之光，已经无情地降临。
沈策孤身一人，站在那片不断坍塌、彻底黑暗的废墟之中，依然在对他微笑。那道没有任何颜色的恐怖光束扫过，将他那修长挺拔的身躯寸寸瓦解，化作了漫天的飞灰，最终，彻底消散在了那片绝对的虚无之中。

“沈策——！”

肖山那绝望、凄厉到了极点的怒吼声，瞬间被狂暴的时空乱流彻底吞噬。
那道深蓝色的空间裂缝，在他的面前，轰然闭合。

**[第十二章·完]**
