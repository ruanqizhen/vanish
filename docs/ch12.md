# 第十二章：高维降临与金丹

## 废墟中的光芒

黑暗的地下空间里，死寂得令人窒息。

肖山站在碎裂的青石碑前，手中紧紧攥着那半截象征着“木偶师”的红色人皮。他能感觉到自己的心跳，在经历了极致的生死一线后，正以一种沉缓而有力的节奏跳动着。

赢了。
他用最暴烈的因果悖论，强行摧毁了这个怪谈的底层规则。

“结束了吗？”肖山低声问道，声音在空旷的地下室里回荡。

沈策没有立刻回答。他握着银刀站在几米外，眉头紧锁，目光死死地盯着刚才厉鬼消散的地方。

“不对劲。”沈策的声音里透出一丝罕见的凝重。

肖山顺着他的目光看去。

在厉鬼溃散的中心位置，并没有像普通的灵异怪谈那样化为飞灰。在那里，有一团极其纯粹的光芒正在缓缓凝聚。

那是一个拳头大小的球体。
它散发着温润如玉的金色光芒，光晕之中，隐约可见无数细小繁复的金色符文在流转、生灭。那是一种与这片阴冷压抑的空间截然不同的气息，浩瀚、古老，仿佛蕴含着一个微型宇宙的磅礴生机。

“那是什么？”肖山走近了几步，能感觉到一股温暖的气流扑面而来。

“厉鬼吞噬了上面那几百人的生命精气，以及这个规则怪谈累积了十年的怨念。在它的底层规则被你强行撑爆的瞬间，这些极其庞大且驳杂的能量无处宣泄，在极致的压力下，发生了坍缩。”

沈策走到那团金光面前，眼神中闪过一丝震惊。
“这东西，在某些古老的修炼体系里，被称为——金丹。”

肖山愣住了。
金丹？修仙小说里那种能让人脱胎换骨的金丹？
他一个写代码的程序员，在这充满赛博诡异风格的规则怪谈里，爆出了一颗修真界的金丹？

这世界的画风是不是有点错乱了？

“它怎么会出现在这里？”肖山伸手想要触碰，却被沈策一把拦住。

“别碰它，里面的能量太狂暴了，你现在的凡人之躯承受不住。”沈策沉声道，“这个世界的底层逻辑比我们想象的还要混乱。这里不仅有系统的代码法则，还混杂着其他更高维度的修炼体系。系统在崩溃时，把无法解析的残余能量，强行封装成了这种形式。”

轰隆隆——

就在两人说话间，整个地下空间突然剧烈地摇晃起来。
这不是普通的地震，而是一种从空间结构内部传来的崩塌感。头顶的青砖开始一块块化作细小的飞灰，簌簌落下。

肖山抬头看去，瞳孔骤然紧缩。

透过逐渐崩解的天花板，他看到了外面的景象。
但那已经不再是他熟悉的林江市了。

## 高维抹杀

夜空变成了深沉的紫黑色，没有星光，没有云层。
而从那无尽的虚空深处，正降下一种无法用人类语言描述的“光”。

它没有颜色，却带着一种绝对的、无可违逆的抹杀意志。光芒扫过之处，地面上的建筑、街道、甚至远处的江水，都在瞬间失去了厚度，变成了一张张扁平的画纸，然后碎裂成漫天的虚无飞灰。

“那是……”肖山只觉得浑身的血液都在倒流。

“高维清洗。”沈策的声音变得极其冰冷，“你摧毁了厉鬼，但也打破了这个区域的规则平衡。天地法则检测到了无法修复的严重异常，它不打算修补了，它要直接把这片区域彻底抹除。”

“它要抹除整个林江市？！”

“不，它的目标是我们。林江市只是被牵连的沙盘。”沈策转头看向肖山，那双一向波澜不惊的眼睛里，第一次露出了决绝的死意。

“没时间了。”

沈策突然出手，一把抓起悬浮在半空中的那颗金丹。
金光触碰他皮肤的瞬间，发出“嗤嗤”的灼烧声，沈策的手掌立刻出现了焦黑的痕迹，但他连眉头都没皱一下。

他猛地转身，用一种不可抗拒的力量捏住了肖山的下巴，将那颗滚烫的金丹粗暴地塞进了肖山的嘴里。

“咽下去！”

“唔——！”肖山瞪大了眼睛，拼命想要挣扎。

金丹入口即化，化作一股滚烫的岩浆，直接冲入了他的腹中。剧烈的灼烧感瞬间剥夺了他的所有感官，他感觉自己的五脏六腑都在被烈火焚烧，经脉在被野蛮地拓宽、撕裂、然后重组。

“啊啊啊啊——！”肖山跪倒在地，痛苦地嘶吼着，皮肤表面渗出耀眼的金色光芒。

“听着！”沈策一把揪住肖山的衣领，将他提了起来，声音在崩塌的轰鸣中依然清晰，“这颗金丹的力量足以护住你的灵魂，不被空间乱流撕碎。这是你唯一活下去的机会！”

“那你呢？！”肖山强忍着剧痛，死死抓着沈策的手臂。

沈策笑了。
这是肖山第一次看到他笑。那个总是带着嘲讽和冷漠面具的执行官，此刻的笑容里透着一种释然与温情。

“我是个局中人，肖山。”沈策轻声说，“我的使命，就是把你送出去。”

光束已经穿透了地下室的防御。
周围的一切都在分崩离析，青砖化作齑粉，空气被压缩到了极致。

沈策松开肖山，从怀里掏出了一样东西。

一块银色的、没有指针的怀表。

他用左手紧紧握住怀表，拇指狠狠地按下了顶端的机括。

咔嚓。

怀表的表面爆发出刺目的深蓝色强光。那光芒带着某种更高维度的力量，在他们面前硬生生撕开了一道空间裂缝。裂缝的另一端，不是正在崩溃的林江市，而是一片深邃狂暴的虚空乱流。

“走吧。”

沈策用尽最后的力气，在肖山的胸口猛地一推。

肖山的身体不受控制地向后倒去，跌入了那道蓝色的空间裂缝中。
在视线完全被裂缝吞没的最后一刻，他看到了令他肝胆俱裂的一幕——

高维的毁灭之光降临了。
沈策站在正在坍塌的黑暗中，微笑着看着他。那道无色的光束扫过沈策的身体，将他的身躯寸寸瓦解，化作漫天的飞灰，消散在绝对的虚无之中。

“沈策——！”

肖山的怒吼声被空间的乱流彻底吞噬。
蓝色的裂缝在他面前轰然闭合。

【第十二章 完】
