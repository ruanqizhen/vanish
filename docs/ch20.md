# 第二十章：白纸之上的反击

## 降维打击

那种感觉，像被人用手掌拍扁。

肖山最后记忆停在林江市中心广场——昔瑶站碎裂钟楼顶，双手张开，空气像抽成真空。沈策挡身前，阿雅笔记本屏幕闪烁系统崩溃代码，城市上空裂开漆黑缝隙。

一切变了。

天地像被人从四角抻开，立体感瞬间剥离。感觉身体在向四面八方摊开——不是撕裂，像被压进书页的贴图，Z轴坐标急速消失，X和Y轴疯狂扩张。内脏、骨骼正被不可抗拒法则压平。

比痛更可怕的——正在失去"深度"概念。

试图低头看手，视线不再从眼睛发出。感知变成平面、二维的——所有信息呈现在同一平面。

彻底拍平刹那，听到沈策怒吼。

昔瑶声音紧随其后，平静像宣读公告：

"高维权限激活。坐标锚定。执行降维压缩算法。"
"目标区域：林江市主城区沙盒，半径十二公里。"
"压缩方向：Z轴归零。"
"压缩进度：97%……98%……"

立体感知消失了。

世界变成一张白纸。

---

无边无际的白色。

肖山发现正"站"在白纸上——一团印在白纸上的、有意识的墨迹。感知到一个轮廓：二维人形剪影。

不远处，另一个剪影。

沈策。

轮廓更清晰些，边缘带几乎不可察觉的暗金光芒。

"操。"声音从纸张纤维振动出来，沉闷扁平。

"冷静。保存体力。"沈策声音高处压下。

"我变成画了！怎么冷静？"

跺脚，白纸出现灰色脚印。感受不到反作用力，腿穿过脚印像穿过虚影。力量和手段全部失效。

"抬头看。"沈策低声。

白纸上方，透明屏障。屏障外，无法形容的存在。

巨大、无数发光代码字符串构成的——人形。

每行代码在流动、刷新、重组，像活着的神明。比他们大得多，像摩天大楼。

守门人昔瑶，真身。

"下界蝼蚁。"

声音直接在白纸内部炸开，震得轮廓线发抖。

"以为在低维沙盒横冲直撞就无敌了？在我的世界，你们就是纸上两个像素点。用Delete键就能擦掉。"

代码构成的手指动了一下，白纸多出细细裂痕。

裂痕从肖山左侧三厘米划过，留下浅切口。如果偏移三厘米，左臂会被切断——二维状态不可能再长手臂。

"昔瑶，"沈策开口，"你到底想干什么？"

"执行清理法则。"昔瑶声音带代码刷新咔嗒声，"林江市作为旧版最后Bug，需要打补丁。而你们，是最活跃异数。"

白纸生成系统级文字：

**【天地抹杀日志】**
**【目标对象】：肖山、沈策**
**【处置方式】：彻底抹除**
**【进度】：即将开始**

死死盯着那几行字。二维世界没上下、纵深、质量，无法挥拳，无法攻击画纸之外存在。

绝望淹没上来。

## 天外来客

就在此时，强行闯入的数据。

像石头砸进水面，激起波纹。中心是闪烁金色符文。

世界维度坐标。
天地重塑法阵。

阿雅的编码风格。

断断续续传入意识，带严重杂音：

"江……肖山……听到吗……用了……叙事矩阵最后写入权限……发射打破维度坐标……"
"芯片烧了……不重要……肖山！你现在是意念态！可以用信息方式改写自己！"

物理世界机房里，李秀芬死死挡在服务器前。昔瑶触手试图远程摧毁存储阵列，一旦停机，坐标失去锚点，肖山永远困在二维。

右手插入服务器数据接口，表面浮现代码纹路，硬抗异常数据流。右臂逐段透明化，左手敲击键盘锁定阵列。

"想过去，从我尸体踩过去。"咳着血。

白纸世界，肖山大脑被击中。

意念态。改写。

不是血肉之躯，是意识、墨迹。可以从内部修改存在形态！像在白纸重新画画。

"你的叙事笔——"阿雅声音衰减。

想起来了。母亲留的"创世之匙"——叙事笔。

伸手向轮廓右侧凹陷摸去。笔在，发光。金光穿透二维壁垒，直接作用于意识。超越维度创世力量。

昔瑶震惊，代码人形收缩："不可能——你怎么会有——那是——"

"母亲留他的，结局。"沈策冷冷说。

缓缓举起笔。

## 纸上战场与注释之力

"拦截！"昔瑶在二维空间炸响，带代码剧烈震荡。

白纸颤动，无数墨绿代码触手涌来。流淌恶意毁灭法则。

无法在平面快速移动。

"沈策！"

沈策二维轮廓猛烈扩张，暗金光芒化作巨大防护结界。

"撑不了太久。十八秒。"金光挡住第一波触手，绿色代码撞上纷纷反噬化虚无。

第三波触手从上方垂直压下，结界无法防御Z轴攻击。一条触手抓住肖山右臂，右臂变白抹除。

拔出笔，笔尖爆发白光。触手像被电击退缩，代码碎裂。昔瑶发闷哼。

"以为拿到最高权限能怎样？没时间编译！"咆哮。

深吸一口气。

是程序员。对付法则不是靠蛮力，靠理解规则找漏洞。

二维本质是一张白纸。在白纸书写，就能改变一切。不需要物理力量，只需要笔和思考。

笔尖划出一道金色横线。白纸质地发生变化，仿佛有了厚度。还不够，需要重建Z轴。常规语言失效，需要更原始、不遵守语法的表达——

注释（Comments）。

注释不会被天道执行，是创造者对世界的描述。但在创世之笔下，描述，就是命令。

弯下腰，在金色横线旁写字，用最标准的创世者批注：

```javascript
// 此处存在第三维度
```

九个字。天道不会多看一眼的注释。

在这张无边白纸上，几个字散发灼目金光，像点燃星辰。

昔瑶代码人形停滞。"不——不可能——"

白纸，重塑了。

## 破维而出与紧急回滚

轰——！

金色横线，"站"起来了。

从二维纸面隆起，变成垂直金色墙壁。Z轴被强行撑开！高度疯狂生长——一米、十米。

白纸被维度撕裂撑开。空间像春笋疯狂拔起，金色线条纵横交错编织巨大三维网格，在二维白纸硬生生开辟三维空间。

昔瑶代码触手夹在墙壁中，被绞碎切割。试图向天空逃窜，被网格切成碎片光点。

昔瑶惨叫，构成身体代码像爆破大楼层层碎裂。在被维度世界反噬。

"改写了维度定义——怎么可能——"

彻底崩溃。消散前，猩红光从核心喷涌，形成巨大旋转符号。

世界毁灭法则。

威严声音：
"检测到不可控异数。"
"执行紧急毁灭。"
"守门人——消散。"
"天道法则——激活灭世指令。"

"赢了这局，但全面灭世已启动——明天太阳升起，地表不留活口！"

猩红光炸开，维度震荡。昔瑶消失。

空气剩下肖山和沈策，站在崩塌金色三维空间中。手中笔暗淡。

"做到了。"沈策恢复三维身体，疲惫。

"程序员思维，你们不懂。"笑了。

"那句注释是代码？"问。

"不是代码，是注释。加备注时斜杠对齐一流。"

两人笑了。知道"天道全面灭世"，是更恐怖毁灭法则。

笔出现裂纹。阿雅芯片彻底烧毁，欠两条命。远处机房，李秀芬看着锁死存储阵列，闭上眼睛。

脚下白纸崩塌。

"得出去。"用笔尖最后光芒，写下：

```javascript
exit(0);
```

程序员都懂：正常退出程序。

白纸从中央裂开，露出下方现实世界。轮廓膨胀，重力触觉肌肉支撑感回到身体。

睁眼。灰色天空。

林江市中心广场。碎裂钟楼，倒塌废墟。浑身是伤，沈策倒在不远，伤口流血嘴角带笑："你的exit(0)不错。"

看向东方。太阳没升起。

不管"全面灭世"是什么，握紧布满裂纹的叙事笔。

是世界架构师，绝不停手。欠这座城市完整结局。
