## 作品简介

《消散的终点》是一部融合都市异能、克苏鲁惊悚、游戏化数值、架空历史、玄幻修仙与硬科幻高维战争的跨维度爆款长篇巨作。

主角肖山，一名底层外卖骑手，在暴雨深夜接到一份指向地图空白区的外卖订单。这场看似普通的配送，将他卷入了一个由"叙事引擎"操控的世界——一个可以被热更新、可以被降维格式化、可以被改写剧本的虚假宇宙。

与冷酷执行官沈策的相遇，让两个孤独的灵魂在无数次死亡回溯中建立了超越生死的羁绊。从破解规则怪谈的程序员思维，到吞噬金丹逆天修仙；从架空大乾王朝的官场博弈，到硬刚高维清理者的宇宙决战——肖山用程序员"卡BUG"的硬核逻辑，一步步撕开世界的真相。

当终点的打字机浮现在眼前，他才明白：真正的自由意志，不是成为世界的神，而是拒绝成为神的资格，选择做一个有血有肉的凡人。


## ✍️ 作者简介：孤鸿

> “他们剥夺了我敲击代码的权力，将我放逐至黑夜与暴雨的街头。可他们忘了，文字是无需编译器的代码，而命运，不过是一场注定被我卡出BUG的剧本。” —— 孤鸿

被读者公认为网络文学史上“身世最离奇、控笔最冷酷、拿了逆天改命剧本的真·传奇神魔”。

### 经历：三起三落的命运过山车
- **南洋碎梦：千亿家产的黄昏**：生于吉隆坡顶级华侨富商之家。八岁那年，父亲在商战中遭遇神秘的“算法金融做空”，猝然离世，千亿家产在两周内被恶意清算殆尽。孤鸿随母亲狼狈回国，寄居在南方小城10平米的潮湿筒子楼里。他守着父亲留下的唯一一块停摆的机械怀表，在煤油灯下完成了对现实逻辑的最初启蒙。
- **车大精英，鹅厂架构师**：凭一股狠劲在煤油灯下苦读，孤鸿一举考入东南大学计算机学院。毕业后杀入头部大厂，仅用五年便主导了服务数亿用户的核心架构，迎娶同厂的“白富美”厂花，攀上时代之巅。然而在高处，他因拒绝配合资本启动一项“剥削全网用户”的黑产算法，反遭高层联手做局构陷。一夜之间，他不仅被开除，更被签署了全行业封杀、赔偿 3000 万天价竞业违约金的死局。
- **街头困兽：年薪百万的黄金骑手**：路被彻底堵死。为了撑起家庭、偿还天价债务，这位曾经在金字塔尖指挥千亿级数据流的顶级架构师，被迫脱下高定西装，换上黄色骑手服。他跨上破旧电动车，成了穿梭在林江市暴雨深夜里最普通的底层骑手。一天 16 小时的机械配送，让他看尽了现实算法对底层劳工的残酷榨取。

### 写作契机
由于竞业协议的存在，孤鸿在现实中被禁止触碰任何与代码开发相关的行为。在被剥夺执编程权的日子里，他做出了一个疯狂的决定——既然无法在电脑里写代码，那就在小说里重构宇宙！

在嘈杂的等餐档口、在夜雨的电动车后座，他掏出屏幕碎裂的手机，用东大计算机系锤炼出的逻辑，把文字当做变量，把怪谈当做BUG，一字字敲出了这部《消散的终点》。他不仅是在写小说，更是在用数百万字搭建一个对抗命运引擎的虚拟系统。 他的横空出世，被主流文学界誉为“2025年人类自由意志在数字囚笼里的终极咆哮”。
