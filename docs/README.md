# 《消失的终点》

> 暴雨深夜，一份不存在的派单。  
> 程序员外卖员江临，踏入了地图上不存在的13号公馆。  
> 从那一刻起，他不再只是送外卖的人——  
> 他成了整个叙事引擎里，唯一能改写结局的变量。

## 作品简介

《消失的终点》是一部融合都市异能、克苏鲁惊悚、游戏化数值、架空历史、玄幻修仙与硬科幻高维战争的跨维度爆款网文。

主角江临，一名底层外卖骑手，在暴雨深夜接到一份指向地图空白区的外卖订单。这场看似普通的配送，将他卷入了一个由"叙事引擎"操控的世界——一个可以被热更新、可以被降维格式化、可以被改写剧本的虚假宇宙。

与冷酷执行官沈策的相遇，让两个孤独的灵魂在无数次死亡回溯中建立了超越生死的羁绊。从破解规则怪谈的程序员思维，到吞噬金丹逆天修仙；从架空大乾王朝的官场博弈，到硬刚高维清理者的宇宙决战——江临用程序员"卡BUG"的硬核逻辑，一步步撕开世界的真相。

当终点的打字机浮现在眼前，他才明白：真正的自由意志，不是成为世界的神，而是拒绝成为神的资格，选择做一个有血有肉的凡人。
