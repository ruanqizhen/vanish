## 作品简介

《消散的终点》是一部融合都市异能、克苏鲁惊悚、游戏化数值、架空历史、玄幻修仙与硬科幻高维战争的跨维度爆款长篇巨作。

主角肖山，一名底层外卖骑手，在暴雨深夜接到一份指向地图空白区的外卖订单。这场看似普通的配送，将他卷入了一个由"叙事引擎"操控的世界——一个可以被热更新、可以被降维格式化、可以被改写剧本的虚假宇宙。

与冷酷执行官沈策的相遇，让两个孤独的灵魂在无数次死亡回溯中建立了超越生死的羁绊。从破解规则怪谈的程序员思维，到吞噬金丹逆天修仙；从架空大乾王朝的官场博弈，到硬刚高维清理者的宇宙决战——肖山用程序员"卡BUG"的硬核逻辑，一步步撕开世界的真相。

当终点的打字机浮现在眼前，他才明白：真正的自由意志，不是成为世界的神，而是拒绝成为神的资格，选择做一个有血有肉的凡人。


## ✍️ 作者简介：孤鸿

> “我曾以为代码能改变世界，后来发现，命运才是最无情的叙事引擎。我送外卖不是为了体验生活，是为了在既定的剧本里，卡出一个活下去的BUG。” —— 孤鸿

被读者戏称为“戏里卡BUG改命，戏外拿了主角剧本”的硬核狠人。

### 经历：从南洋豪门到街头骑手
* **南洋旧梦，家道中落**：生于吉隆坡华侨富商之家。八岁时父亲离世，家族企业骤遭恶意清算，千亿家产倾刻消散。随母回国寄居筒子楼，尝尽冷暖。
* **车大精英，大厂巅峰**：凭一股狠劲在煤油灯下苦读，最终考入东南大学计算机学院。毕业后赶上互联网浪潮，凭借算法天赋一路高歌，五年内晋升为头部大厂核心架构师，并迎娶同厂厂花，攀上人生巅峰。
* **时代一叹，破产外卖**：行业寒冬突袭，被迫“毕业”。随后因激进创业投资遭遇黑天鹅，背负巨额债务。为撑起家庭，他脱下西装换上黄色骑手服，跨上电动车，成了穿梭在暴雨深夜里的外卖骑手。

### 写作契机
这段“从高空坠落”的经历，成为他最锋利的创作武器。在嘈杂的等餐档口、在夜雨的电动车后座，他掏出破旧手机，将系统架构思维、克苏鲁神话与街头规则怪谈完美缝合，用冷酷的算法逻辑拆解荒诞命途，一字字敲出了这部《消散的终点》。他的横空出世，被文学界与网文圈共同誉为“2025年工业化叙事的最高峰”。
