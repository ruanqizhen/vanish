# 第十八章：没有尽头的绝望

## 末日降临

昔瑶笑了。

那一抹笑容冷冽而空灵。她端起那只古旧的青瓷保温杯，慢条斯理地拂去杯口萦绕的雾气，那姿态仿佛一位高高在上的神明，正静静地俯瞰着被她掌控的夜色。

在她的背后，整座林江市的天空正在发生诡异的“扭曲”。

那不是被某种粗暴的外力撕裂，更像是一幅正在腐烂的画卷，从最薄弱的肌理处开始自行溃散。厚重的云层向内疯狂坍塌，暴露出令人极度不安的深紫色高维虚空。那片虚空中没有任何星辰的闪烁，只有沉重且高密度的绝对黑暗在缓慢翻滚着，仿佛一头正在苏醒的灭世巨兽。

肖山下意识地想要移动，却骇然发现自己的身体已经被系统级权限强行锁死。脚下的触感变得异常柔软且黏稠，坚硬的柏油马路竟然融化成了深黑色的液态泥沼。泥沼中，有无数细小的光点在游动，就像是某种不可名状的高维蛊虫。

“小肖啊。”昔瑶的语气清冷中甚至带着一丝温柔的叹息，她缓缓将手中的保温杯倾斜，“你帮我跑了那么多单外卖，我一直想找个机会好好谢谢你呢。”

从杯口倒出来的根本不是水。
而是纯粹的“虚无”。

那是一种能够绝对吞噬光线的黑色，如同实质般的瀑布倾泻而下。周围路灯的光源瞬间被“抹去”，坚固的钢筋混凝土建筑如同接触到强酸般迅速溶解，整个林江市在这股黑色的侵蚀下，变得模糊且极度扭曲。

呼吸变得异常困难。空气中的恶意指数正在呈几何级数急剧飙升。

“别白费力气了。”昔瑶微笑着，看向正试图强行调动真气反抗的沈策，“沈策，作为一个在沙盒里被重置了无数次的异常变量，你能硬生生扛到现在，确实不容易。但这里是林江市，是我的——”

她顿了顿，似乎在仔细品味着这个词汇的分量。

“——终极之地。”

话音落下的瞬间，整个世界陷入了绝对的死寂。

肖山下意识地看向远处的街道——刚才还在惊慌失措四处逃窜的行人，此刻全都定格了。
他们依然保持着奔跑的姿势，但眼睛里已经失去了所有鲜活的光芒，空洞得如同被抽走灵魂的木偶。

一个中年女人僵立在便利店门口，嘴唇像机械般机械地张合着，反复重复着同一句毫无起伏的话：“孩子还在幼儿园我要去接他孩子还在幼儿园我要去接他……”

没有任何情绪的起伏，就像是一段陷入了死循环的、没有尽头的梦魇代码。

不仅仅是她。
整条街道上的人，都在如同卡带般，永无止境地重复着生前最后一刻的动作。

“他们没死。”昔瑶的声音透着一种令人毛骨悚然的怜悯，“只是陷入了底层逻辑的轮回。一遍又一遍，永远无法解脱。你觉得，这算不算是比直接杀死他们更残忍的恩赐？”

“你他妈——”肖山目眦欲裂，刚想拼尽全力扑过去，那股冰冷的黑色气息却已经漫过了他的脚踝。

昔瑶缓步走到肖山面前。清脆的高跟鞋声在融化的马路上敲击出空灵的回响。她终于彻底褪去了所有的伪装——她不再是那个唯唯诺诺的站点调度员，而是俯瞰了这座城市三十年的、冷酷无情的神明。

“你是不是在想，为什么偏偏是你？为什么偏偏是林江市？”昔瑶的指尖在虚空中轻轻划过，流淌出几根发光的系统丝线，“因为我在这里，足足编织了三十年。三十年前，林江市不过是个破落的小镇。现在这里有多少人？两百三十万。这些多出来的生命——全都是我一行一行代码‘造’出来的。”

“这里的每一根丝线，就是他们所谓的命运。我精密地安排着每一次出生、每一次相遇。我把林江市写成了——”她冷冷地看向沈策，眼底闪过一丝复杂的波动。

“——一个完美的测试场。我把这些底层法则称为‘天道法则’。一共有十三条。你们所经历的世间百态——根本不是什么世界规则，而是我给你们量身定制的牢笼。而你，肖山——”

她微微俯下身，眼神狂热。

“你是第一个，在所有十三重牢笼里活下来，并且看破虚妄的异数。”
“所以，你将成为献给这片天地的、最完美的祭品。”

## 绝境与混沌

地面开始发生剧烈的震颤。

沈策双手死死地撑着地面，额头上的青筋如虬龙般暴起。他的嘴唇飞速开合，正在念诵着某种古老而晦涩的言灵咒语。缠绕在他们周围的黑色气息，在这咒语的冲击下发出尖锐刺耳的嘶鸣。

“肖山——！”沈策的双眼重新燃烧起璀璨的金光，他咬着牙嘶吼道，“我撑不了多久！快找系统的退路！”

昔瑶微微挑了挑眉，似乎有些意外：“上古言灵咒语？这种陈旧的补丁代码，居然还能在我的天地里生效。你的灵魂强度，确实比我想象的还要顽强。”

她随手打了个响指。

沈策的胸口如遭重锤，整个人如同断线的风筝般倒飞而出，硬生生撞弯了一根路灯杆。他剧烈地咳嗽着，咳出的鲜血中竟然夹杂着闪烁的逻辑光点。但他依然死死地咬着牙，挣扎着重新站起，继续念诵着那古老的咒语。

“没用的。言灵的本质是向天道祈求权限，而在这里——我，就是天道本身。”

她缓缓张开双臂。

天空彻底裂开了。暴露出那不可名状、正在疯狂翻滚蠕动的高维混沌。无数扭曲的光影在其中纠缠、撕裂、重组，形成了一幅足以令人瞬间精神崩溃的恐怖图景。

肖山突然感到鼻子一热，殷红的鲜血滴落。人类的大脑根本无法解析这种完全超越维度的恐怖信息。

“放弃吧。人类那孱弱的神经元，根本无法承受高维的真相。”

肖山的思绪陷入了极度的混乱，理智在信息过载的边缘摇摇欲坠，即将全面崩溃。

就在这千钧一发之际，他猛地低下了头，视线落在了腰间那个破旧的外卖箱上。

在这绝对的黑暗中，外卖箱里竟然隐约有光芒在闪烁。那光芒没有被深渊同化，而是散发着一种绝对独立的、不受任何系统规则约束的、不可篡改的纯粹光芒。

“嗯？”昔瑶敏锐地察觉到了异常，“那个破箱子居然能发光？这种底层的权限代码……根本不是属于这个世界的东西。”

她停下脚步，眼神变得无比锐利：“那是谁给你的？”

## 十秒轮回

时间突然开始倒流。

原本滴落在地上的雨水，如同违背了物理法则般，诡异地倒飞回漆黑的夜空中。
周围那些陷入死循环的行人，动作也开始像录像带倒放般向后退去。

一切似乎都在重新开始。

然而，仅仅循环执行了十秒钟，整个世界便再次发生回退。

“这也是我的法则之一：时光回溯。”昔瑶用一种居高临下的淡漠语气说道，“每一段逻辑每十秒就会强制重置一次。在被彻底抹杀之前，你们可以尽情地体验这种无休止的绝望。”

肖山感到一阵头皮发麻。只有十秒，十秒钟后一切就会被强制重置归零。这是一个根本找不到任何出口的绝望轮回！

在经历了无数次短暂而痛苦的轮回后，肖山的记忆已经开始出现混乱的重影，但他依然死死地盯着腰间的外卖箱。他发现，外卖箱里的那团光芒，在时间疯狂回退的过程中，竟然始终保持着不变的亮度。

它完全不受时光回溯法则的影响！

这是老黄留给他的箱子。那个在林江市默默无闻地跑了八万单外卖的老黄，在彻底消失前，郑重其事地塞给他的破旧箱子。

又一个十秒的死亡循环重新启动。

就在时间重置的那个绝对零点一秒的间隙，肖山毫不犹豫地将手伸进了箱子里。

## 尘封的历史

他的指尖触碰到了箱底一层薄薄的东西。就在接触的瞬间，一股不受系统限制的庞大记忆流，如同破堤的洪水般疯狂窜入他的大脑。

他看到了十五年前的林江市。

年轻时候的老黄，骑着一辆破旧的电动车，停在了老城区一栋筒子楼前。他气喘吁吁地爬上三楼，敲响了307室的门。

门开了。年轻的母亲林知意，怀里正抱着还在襁褓中的婴儿肖山。

老黄将一份外卖恭敬地递上，在雨夜中用那略显笨拙的手迹写下了一行字：“以后这单，不管多远、多晚，我都自己送。”

在肖山的意识被狠狠扯回现实之前，他看到了老黄在那个大雨倾盆的夜晚，郑重地将那个外卖箱交接出去时的画面，还有他那句在雨声中显得无比清晰的承诺：
“告诉他，307室的门，永远是为他开着的。”

肖山猛地回过神来，他的手中已经多了一张泛黄的外卖单据。单据的边缘沾着早已干涸的暗褐色血迹，那是未被岁月和系统抹去的、绝对真实的痕迹。

在这张薄薄单据的最下方，赫然写着两行熟悉的、手写的签名：

林知意。
肖海川。

## 时光裂缝

“这怎么可能……”肖山的手在剧烈地发抖。父母的亲笔签名，怎么会出现在一张十五年前的旧外卖单上？

这张单据，是这个世界上唯一未被系统格式化的岁月痕迹！

刹那间，那张泛黄的单据爆发出了极其耀眼的光芒。昔瑶那引以为傲的绝对黑暗领域，在这道光芒面前，竟然像初雪遇到烈日般被迅速撕裂融化！

“这——这不可能！”昔瑶那张始终波澜不惊的脸上，第一次露出了惊怒交加的表情。

单据上的血迹逐渐变得鲜红欲滴，从光芒的深处，竟然传来了母亲林知意那跨越了时空的声音：“小临……来十年前……”

那是跨越了整整十年时空的呼唤！

“沈策！”肖山爆发出嘶哑的怒吼，“抓住我的手！”

沈策看到了那道撕裂绝望的光芒，没有任何犹豫，拼尽全力挣脱了规则的束缚，朝着肖山狂奔而来。

地面轰然炸裂，无数条粗壮的黑色巨型触手带着毁灭的气息，从深渊中疯狂席卷而来，试图将他们彻底碾碎。

“你以为凭一张废弃的系统单据，就能从我的眼皮底下逃走吗？！”昔瑶暴怒咆哮。

肖山双手死死抓住那张单据，然后，向着两边狠狠一撕！

“嗡——！”

这不仅仅是撕碎了一张纸，而是在天道绝对严密的法则中，生生地撕开了一道时光的裂缝。刺目的白光轰然爆发，那些触碰到白光的黑色触手，瞬间被底层逻辑判定为“非法入侵”，直接溃散成了虚无的代码。

“不——！”昔瑶发出了极度恐慌和不甘的咆哮。

肖山一把反握住沈策的手腕，两人毫不犹豫地跃入了那团耀眼的光芒之中。

他们迅速坠入了一条由无数光点构成的时光通道。四周的画面开始以一种疯狂的速度倒退，就像是一幅被按下了高速倒放键的历史画卷。

沈策那冰冷的手指，在光芒中用力地回握住了肖山。

“怕你个傻子又走丢了。”他在呼啸的光流中轻声说道。

他们正向着十年前的林江市坠落。那是改变了一切悲剧的过往，也是所有因果的源头。

刺目的白光终于缓缓散去。

两人艰难地睁开眼睛。
这是一个无比普通的夏日夜晚。耳边传来阵阵聒噪的蝉鸣，空气中甚至还弥漫着烧烤摊那种特有的、烟火气十足的孜然味。老旧的居民楼里，亮着星星点点温馨的灯光。黑夜并没有像刚才那样试图吞噬一切，人们依然在这座城市里过着真实而庸常的生活。

他们真的回到了十年前。

“这……”沈策半跪在地上，气息依然有些微弱。

“十年前的林江市。”肖山的声音因为极度的激动而微微发抖，“我们真的回到了过去。”

十年前的林江市，正沉睡在这个宁静的夜色中。
而他们，将要在不可挽回的毁灭降临之前，在这段尘封的岁月中，找到改变一切既定命运的源头。
