# 第十八章：没有尽头的绝望

## 末日降临

昔瑶笑了。

那是一种从骨子里透出来的、冷冽而空灵的笑声。她端起那只古旧的青瓷保温杯，慢条斯理地拂去杯口的温热雾气，仿佛一个高高在上的神明，正静静地看着下班的夜色。

可她的背后，整座林江市的天空正在发生“扭曲”。

不是被什么外力撕开，而是像一块腐烂的画卷，从最薄弱的地方开始自行溃烂。云层向内坍塌，露出后方令人不安的、深紫色的虚空。那虚空之中没有星辰，只有一种沉重的、高密度的黑暗气息在缓慢地翻滚，像是一只正在苏醒的巨兽。

肖山想要动，却发现自己的身体被强行锁死了。脚下的地面变得柔软而黏稠，柏油马路正在融化成深黑色的液态泥沼。那些细小的光点在里面游动，像是某种不可名状的蛊虫。

“小肖啊。”昔瑶的语气依然是那副清冷中带着温柔的叹息，她将那只青瓷保温杯微微倾斜，“你帮我跑了那么多单外卖，我一直想谢谢你。”

保温杯倾斜。倒出来的不是水。

是纯粹的“虚无”。

绝对的、吞噬一切光线和希望的黑色，像实体一样从杯口倾泻而出。所过之处，路灯的光源被直接“抹去”，建筑被溶解，林江市在这个黑色的侵蚀下变得模糊、扭曲。

肖山感到呼吸变得困难。空气中的恶意在急剧增加。

“没用的。”昔瑶微笑着看着正在试图强行反抗的沈策，“沈策，你作为一个被重置了那么多次的异数，能扛到现在很不容易。但这里是林江市，是我的——”

她顿了顿，似乎在品味这个词。

“——终极之地。”

话音落下的瞬间，整个世界安静了。

肖山下意识看向街道两旁——那些刚才还在惊慌逃窜的行人，此刻全都停了下来。

他们保持着奔跑的姿势，但眼睛里失去了最后的光芒，空洞得像失去了灵魂的木偶。

一个中年女人站在便利店门口，嘴唇一张一合，反复说着同一句话：“孩子还在幼儿园我要去接他孩子还在幼儿园我要去接他……”

她的声音没有起伏，像一个没有尽头的梦魇。

不止是她。

街道上所有的人都在重复生前最后一刻的动作。

“他们……还没有死。”昔瑶的声音透着一丝怜悯，“他们只是陷入了轮回。一遍，又一遍，永远。这是不是比直接杀死更残忍？”

“你他妈——”肖山想要转身扑过去，但黑色的黑暗气息已经漫到了他的脚踝。

昔瑶缓步走到肖山面前。她的高跟鞋在融化的马路上踩出清脆的回响。她褪去了所有伪装——不再是那个温柔的调度员，而是一个俯瞰了这片天地三十年的神明。

“我知道你想问什么。为什么是我？为什么是林江市？”昔瑶指尖在空中轻轻一划，一道发光的丝线流淌出来，“因为我在这里编织了三十年的网。三十年前，林江市只有一个极小的小镇。你知道现在有多少人吗？两百三十万。这些多出来的生命——都是我‘造’出来的。”

“每一根丝线，就是一个人的命运。我安排他们的出生、成长、相遇。我把林江市写成了一座——”她看向沈策，眼底闪过一丝复杂的波动。

“——完美的测试场。你知道这些年来，我管这些法则叫什么吗？‘天道法则’。一共十三条，每一条都是一个可以随时替换的法则。世间百态、生老病死、悲欢离合——它们不是世界规则，是我给你们装载的不同版本的牢笼。而你，肖山——”

“你是第一个在所有牢笼之间活下来，还能看破虚妄的异数。”

“——献给这片天地的祭品。”

## 绝境与混沌

地面猛然震动。

沈策双手撑地，额头青筋暴起，嘴里念诵着某种古老的咒语。那些缠绕他的黑色气息纷纷发出尖锐的嘶鸣。

“肖山——！”他双眼燃烧着金光，“我撑不了多久！找退路！”

昔瑶微微挑眉：“有意思，上古的言灵咒语，居然能在我的天地里生效。看来你的力量比我想的要强。”

她轻轻打了个响指。

沈策的胸口像是遭遇了重锤轰击，整个人倒飞出去，撞弯了路灯。他咳出带光点的血，却挣扎着站了起来，继续念诵。

“没用的。言灵的本质是向天道祈求力量，而在这里——我就是天道本身。”

她张开双臂。

天空裂开了。露出的不是宇宙星空，而是一团不可名状的、不断翻滚蠕动的混沌。无数扭曲的光影在天空中纠缠、重组，形成足以让人精神崩溃的恐怖景象。

肖山感到鼻子一热，鲜血流了出来。他的大脑无法解析这种超越凡人认知的信息。

“放弃吧，小肖。人类的意识无法承受这种恐怖。”

肖山的思绪开始混乱，就在他的理智即将因为过度负荷而崩溃时——

他猛地低头，看向自己腰间那个破旧的外卖箱。

那个箱子里，有一种隐约的光芒在闪烁。它没有被昔瑶的黑暗同化，它在散发着一种独立的、不受系统约束的绝对不可篡改的光芒。

“嗯？”昔瑶察觉到了什么，“一个普通的破箱子，居然能发光？不对……这不是属于这个世界的东西。”

她停了下来，眼神锐利：“是谁给你的？”

## 十秒轮回

时间开始倒流。

肖山眼睁睁看着地上的雨水倒飞回空中，整个世界像是在执行时光倒流。

行人们的动作开始倒放。

然后，一切重新开始。

循环执行十秒，然后再次回退。

“这也是我的法则之一：时光回溯。”昔瑶淡淡地说，“每十秒重置一次状态。你们可以尽情体验绝望。”

肖山的头皮发麻。十秒，只有十秒的时间，一切就会被强制重置。这是一个没有出口的绝望轮回。

在无数次轮回中，肖山的记忆开始混乱，但他死死盯着那个外卖箱。他发现，外卖箱的光芒在时间回退时，依然保持着不变。

它不受时光回溯的影响！

这是老黄留给他的箱子。那个骑了十五年电动车、送了八万单外卖的老黄，消失前塞给他的。

又一个十秒循环开始了。

肖山在时间重置的瞬间，抓住了那零点一秒的间隙，把手伸进了箱子里。

## 尘封的历史

箱底有一层薄薄的东西，当肖山的指尖碰到它时，一股不受限制的记忆猛地窜入他的大脑。

他看到了十五年前的林江市（尘封的过往）。

年轻的老黄骑着电动车，停在老城区的居民楼前。三楼，307室。

门开了，年轻的母亲林知意抱着还是婴儿的肖山。

老黄递上外卖，在雨中站了很久，写下一行字：“以后这单不管多远我都自己送。”

在肖山被强行扯回现实前，他看到老黄在雨夜交出了箱子，说：“告诉他，307室的门永远是开着的。”

肖山猛地回过神，发现自己从箱子里抽出了一张泛黄的外卖单。单据上沾着干涸的血迹，那是未被岁月抹去的痕迹。

而在单据下方，有两行手写签名：

林知意。
肖海川。

## 时光裂缝

“不可能……”肖山的手在发抖。他父母的签名，怎么会出现在十五年前的外卖单上？

这张单据，是一个未被昔瑶抹除的岁月痕迹！

单据爆发出耀眼的光芒，昔瑶的领域在这光芒面前，像遇到烈日一般被撕裂！

“这……这是——”昔瑶惊怒交加。

单据上的血迹重新变得鲜红，光芒中传来了母亲的声音：“小临……来……来十年前……”

这是跨越时空的呼唤！

“沈策！”肖山嘶哑地喊道，“抓住我的手！”

沈策看到了光芒，没有任何犹豫，挣扎着跑来。

地面炸裂，昔瑶召唤出无数条黑色的巨型触手，向他们席卷而来。

“你们以为一张废弃单据就能从我这逃走？”

肖山双手抓住单据，用力向两边一撕！

“嗡——！”

这不仅仅是撕裂纸张，这是在天地的法则中撕开了一道时光裂缝。刺目的白光爆发，那些黑色触手在接触光芒的瞬间被判定为天威，纷纷溃散。

“不——！”昔瑶发出了真正的恐慌咆哮。

肖山一把抓住沈策的手腕，跃入光芒之中。

他们坠入了一条由无数光点构成的时光通道。四周的画面飞速倒退，像是在高速倒放的历史画卷。

沈策的手指用力回握住他。

“怕你又走丢了。”肖山轻声说。

向着十年前的林江市。那个改变一切的过往。

白光散去。

肖山睁开眼睛。

这是一个普通的夏夜，蝉鸣阵阵，空气中带着烧烤摊的烟火气，居民楼里亮着零星的灯光。黑夜还没有完全吞噬，人们还过着真实而庸碌的生活。

他们回到了十年前。

“这……这是……”沈策气息微弱。

“十年前的林江市。”肖山的声音在颤抖，“我们回到了过去。”

十年前的林江市，正在夜色中沉睡。

而他们，将要在世界毁灭之前，找到那个改变命运的一切的源头。

——【第十八章 完】——
