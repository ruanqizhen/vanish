# 第八章：冷掉的黄焖鸡与千亿财阀

## 坠落与重置

天空裂开了。

林江市金融区上空，被霓虹染成紫色的天幕，毫无预兆地被一道银白数据裂缝撕裂。
伴随气压爆鸣，两道身影如断线风筝般跌出，被无形大手抛向现代都市。

失重感让肖山胃部翻江倒海。
狂风灌满残破的外卖服。他睁开眼，下方是密集的摩天大楼，车流如光河。在三百米高空俯瞰，城市像一块巨大的精密主板。

不是大乾王朝。
不是仙门山巅。
这是现代，林江市——他们最初被卷入循环的坐标。

肖山在空中艰难扭转身躯，死死锁定极速下坠的那道身影。
“沈策！”

狂风撕碎了嘶吼。
沈策状态极糟。在大乾时空崩塌最后，他为护肖山，硬扛了高维清理者的抹杀。墨色长袍绞碎成布条，浑身浴血，彻底失去意识。

太远了，抓不住。
即将以坠亡结束穿越时，异变陡生。

下方，景华五星酒店露天平顶，强光探照灯骤亮。雪白光柱直冲云霄，编织成光网，将下坠的沈策稳稳托住。紧接着，酒店上空荡起半透明六边形能量盾，犹如巨兽，将沈策拉扯进去。

光盾闭合，沈策消失。

肖山还没来得及反应，后背狠狠撞上缓冲气垫。巨大冲击力将他弹向半空，翻滚数周后，一头栽进景观喷泉池。

“哗啦——”
冰冷池水灌入口鼻。他从水池中挣扎起身，浑身湿透，血水与泥浆顺着外卖服滴落，像条流浪狗。

抹了把脸，抬头看向金碧辉煌的景华酒店。直觉告诉他，沈策在里面。
突然，口袋手机发出尖锐震动。

掏出布满裂痕的旧手机。
屏幕自动切换到了跑外卖的接单系统。原本的界面被猩红代码覆写，一条置顶加急订单疯狂闪烁：

> **【系统提示：高维强制派单——不可取消、不可转单！】**
> **【订单编号：SJ-19910724-001】**
> **【配送物品：指定商品（商家已备好）】**
> **【取货地址：老城区·钟楼街·小四川黄焖鸡米饭】**
> **【送达地址：金融中心·景华酒店·顶层宴会厅】**
> **【收件人：沈氏集团新任总裁·沈策】**
> **【配送费：¥ 999,999.00】**
> **【警告：限时30分钟内送达。超时将触发抹杀程序。】**

肖山死盯着“沈策”和“沈氏集团总裁”，眼底闪过冷意。
六位数的配送费。
千亿财阀继承人。
高维系统对大乾时空格式化结果不满意，直接在这个现代沙盒里，给沈策安排了全新的人设剧本。

“刺啦——”
耳机切入静电杂音，一个带着浓重黑眼圈质感、语速极快的年轻女声强行黑进频道：
“肖山！你这码农终于掉线重连了！”

“阿雅？”肖山一愣。
常年蜗居地下室，靠几台破电脑硬刚高维系统的黑客少女。

“没时间废话！”阿雅那边的敲击声像在打仗，“你们刚掉进林江市，城市底层数据就重构了！系统抹除了沈策的记忆，注入了‘财阀独子’的伪造履历！他现在正在景华参加继承晚宴！”
“我看到了，系统给我派了个单。”肖山抹去屏幕水珠。
“那是清除诱捕机制！”阿雅声音焦急，“整个景华酒店被高维规则包裹，成了‘规则怪谈’核心区！安保全是数据强化的杀戮机器，你肉身闯进去就是送死！”

“我必须去。”肖山咬牙，摸了摸胸口黯淡的大乾通宝，“他替我挡了格式化，我得把他拉回来。”
“你拿什么拉？用爱感化吗？”
“用黄焖鸡。”
“……”阿雅沉默两秒，“疯了？”

“记忆可以重写，但物理感官的底层锚点很难清洗。那是规则盲区。”肖山翻出绿化带，目光锁定街角一辆生锈的共享电动车，“味觉，是唤醒他的唯一后门。”

“行，算你有理。”阿雅深吸气，“既然系统给了外卖单，我就能利用‘最高优先权’开挂。刚黑进了你手机的‘破维芯片’，现在强行超频。这会把物理碰撞体积以数据网格投射到你视网膜上。但记住，超频只能撑二十分钟，芯片烧毁前必须冲进去！”

“谢了，欠你顿烧烤。”
“你欠我一块四万的主板！活着回来！”

通讯掐断。
肖山跨上破旧电动车。
双手握把瞬间，强烈电流直冲大脑！

> **【破维超频·激活】**
> **【赋予临时特性：绝对闪避、动能无视、路径最优解】**

“嗡——！”
伴随电机咆哮，这辆限速25码的老旧电动车，后轮在柏油路摩擦出焦糊白烟，轰然冲入林江市车流。

## 极限配送

在破维芯片超频下，眼前的世界变了。

复杂的街道、车辆、人群，褪去色彩，化作蓝色数据线和多边形立体线框。
密密麻麻的数据模型中，一条发着绿光的“最优路径”，在车流夹缝中蜿蜒前伸。

这就是外卖员的绝对特权——路线最优解。

肖山压低身体，贴在车把上。
限速？不存在。
红灯？给正常人看的。

他以超120码的时速，在晚高峰主干道上疯狂穿插。
前方，两辆并排公交车的缝隙急速收缩。肖山没有减速，猛扭车头，电动车近乎六十度倾斜，前轮贴着左车皮，后视镜擦着右广告牌，毫厘之差“切”了过去！

公交司机眼前闪过黄色残影，吓得一脚踩死刹车：“这送外卖的不要命了？！”

肖山听不见。
狂风刮过脸颊，双眼死盯视网膜的倒计时。
二十分钟。

不到十分钟，他暴力冲入钟楼老街。
小四川黄焖鸡米饭的招牌闪着廉价红光。老板刚把打包好的餐盒放在台面，就看到电动车带着刺耳刹车声精准停在门口，轮胎拉出长长焦痕。

肖山一把抓过外卖袋，连“谢谢”都被吞没在引擎轰鸣中。
指尖传来滚烫温度。
这熟悉的重量和塑料袋触感，成了疯狂世界里唯一的真实。

“抓紧了，沈策。”
肖山将外卖塞进保温箱，猛拧电门。
车头抬起，电动车发出嘶吼，朝金融中心的景华酒店发起最后冲锋。

## 破门而入

景华酒店广场。

这座顶级地标，此刻已武装到牙齿。
数百名穿黑色战术防弹衣的安保将正门围得水泄不通。眼神机械，耳麦闪红光——他们是被系统临时刷新的“规则守护者”。

“各单位注意，目标接近。”
安保队长的对讲机传来电子音。

街道尽头，缠绕着蓝色数据流光的电动车狂飙而来。

“开火阻截。”

没有任何警告，外围十名安保直接拔枪，黑洞洞的枪口对准那抹明黄。
“砰砰砰砰——！”
枪声撕裂夜空。

然而，在肖山视网膜上，子弹轨迹被放慢成了红色虚线。“配送不可中断”的法则具象化！
他甚至没刻意躲。
电动车在枪林弹雨中以S型扭动。子弹擦着头盔、衣服飞过，无一命中。每次转向都精准卡在弹道盲区，仿佛能预知落点。

“见鬼了！”安保队长瞳孔地震，“收缩防线！结人墙！”

几十名壮汉迅速组成密不透风的肉墙，挡在大理石台阶前。
肖山看着人墙，眼神燃起暴戾。

“给我破！”
距离不到十米，肖山猛捏前刹，身体借惯性向上拔起！
整辆电动车以后轮为轴，车尾高翘，化作重锤，狠狠砸向人墙最薄弱节点！

“轰——！”
几名安保被撞飞，防线撕开缺口。
肖山在车辆脱手瞬间，借力腾空，双脚精准踩在安保肩膀上，凌空越过防线。

没走正门。
正门大厅布满红外切割网。
落地前，双手一把抓住了二楼延伸出的水晶雨棚边缘。手指传来钻心剧痛，咬死牙关，硬生生将身体拉上雨棚。

三楼。四楼。五楼。
在名流震惊目光中，穿着外卖服的疯子像壁虎一样在玻璃幕墙外徒手攀爬。

第八层。
顶层宴会厅巨大落地窗前。

肖山悬挂在八十米高空，右手扣着窗沿，左手从背后抽出那袋黄焖鸡。
隔着防弹玻璃，看向内部。

金碧辉煌，衣香鬓影。
核心主桌旁，穿着银灰色高定西装的男人正举着香槟，接受权贵谄媚。
沈策。

肖山眼底闪过狠厉。
抬起磨损的运动鞋，对准那面号称狙击枪打不穿的防弹玻璃，将“破维超频”最后一丝能量集中在脚尖。

“您的外卖——到了！”

“咔嚓！”
伴随爆裂声。
防弹玻璃在规则冲突下，犹如脆弱冰层轰然粉碎！

玻璃碴在吊灯下折射出刺目光芒，犹如一场晶莹的暴雪。
这场暴雪中，明黄色的身影带着一身寒气与血腥，提着十几块钱的黄焖鸡，犹如战神般，轰然砸落在千亿财阀的晚宴中央。

全场死寂。

**[第八章·完]**
