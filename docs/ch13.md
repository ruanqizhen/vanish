# 第十三章：毁灭之光与跨界雷劫

## 世界的橡皮擦

失重感消失的瞬间，肖山发现自己没有落在坚硬的地面。

他悬浮在半空。

狂风如刀刮过脸颊，冰冷空气灌满肺部。他低头，瞳孔瞬间缩成针尖。

脚下，是林江市璀璨夜景。万家灯火如同倒悬星河，街道车流如织，红色尾灯拉出长长光轨。

被沈策撕开的空间裂缝，直接传送到了距地面上千米高空。

"沈策……"肖山心脏像被无形的手死死攥住。摸向胸口，隔着衣服感受到那块冰冷怀表。沈策用命，把他送出了坍塌的地下空间。

但危机没有解除。
反而扩大到了令人绝望的量级。

肖山看到，新城区方向，一道无法用语言描述的"光束"正从天而降。

不是普通的光。它没有颜色，超越人类视网膜解析极限。像一把从无尽高空垂落的巨大橡皮擦，无声无息抹过繁华城市。

毁灭之光。

光束扫过，一栋四十七层摩天大楼不到零点三秒内失去厚度。雄伟的钢筋混凝土建筑，瞬间压成没有厚度的扁平纸片，像一张贴在空气中的巨型贴画。

紧接着，"贴画"开始瓦解。

从边缘开始，扁平平面剥落成硬币大小的碎片。每一块碎片正面保留着大楼局部画面——一扇亮灯窗户、一个惊恐的脸、半块广告牌。碎片在半空翻滚，最终化为彻底的虚无。

"它在……抹除整个世界？"肖山浑身冰冷。

他终于明白沈策那句"天地不允许异数存在"是什么意思了。因为他这个无法处理的异数，天地启动最高抹杀法则。它不打算抹平变数，要直接把世界彻底重置！

下方街道，抹除灾难疯狂蔓延。

一辆行驶的公交车被光束扫中。前半截车身保持惯性向前冲，后半截已被压成扁平图案。乘客来不及惨叫，身体随着车厢被剥离厚度，变成定格惊恐表情的纸片，碎裂成漫天飞灰。

"停下！"肖山在千米高空绝望怒吼，"冲我来啊！"

吼声被狂风撕碎。但他体内的那颗金丹，这一刻产生感应。

被强行吞下的金丹，原本安静悬浮在胃里。当肖山情绪剧烈波动，对这座被屠杀的城市产生强烈因果牵绊时，金丹爆发了。

轰——！

一股滚烫的金色洪流从丹田炸开。

不是之前撕裂经脉的剧痛，而是浩瀚无垠的纯粹能量。金丹力量沿着血管神经疯狂蔓延，皮肤表面浮现一道道耀眼金色符文。符文交织成复杂因果网络，将他牢牢固定在高空。

同一秒。
天空，变了。

## 杀毒雷劫

林江市正上方的夜空，突然被一片浓如泼墨的乌云覆盖。

不是天气变化。乌云内部，没有水汽，只有密密麻麻疯狂滚动的金色符文。符文以每秒数万行速度流转，像正在酝酿的灭世天劫。

"检测到天地异数。威胁等级：极度危险。启动抹杀法则——"

冰冷毫无感情的机械合成音，在肖山意识深处炸响。

不是人类声音。是天地法则的无情宣告。

天地发现了他。金丹的爆发，让他在天地的感知中变得像太阳一样耀眼。

毁灭之光暂停扫荡，乌云中的金色符文疯狂汇聚，锁定肖山正上方。

"第一重天劫：形神俱灭。"

咔嚓！

水桶粗细的金色雷霆，从云层悍然劈落。

通体半透明，内里流动着亿万行细密的毁灭法则。劈落瞬间照亮整个林江市夜空，仿佛将天幕硬生生撕开裂缝。

肖山根本来不及躲避。在锁定的雷劫面前，逃避失去意义。

金色雷霆精准轰击在肖山身上。

"呃啊啊啊啊——！"

肖山发出凄厉惨叫。

这不是物理电击，而是数据层面的"删除"。雷霆触及皮肤瞬间，右半边肩膀直接消失。没有流血，没有焦黑，肌肉和骨骼像被无形巨手生生抹去，化作光点消散。

超越神经承受极限的剧痛。

但下一秒，体内金丹疯狂旋转。金色灵力洪流涌向消失的肩膀，正消散的光点被硬生生拉回，血肉重新交织，骨骼肉眼可见地再生缝合。

不到三秒，肩膀恢复如初。

肖山大口喘着粗气，额头青筋暴起。看着完好的肩膀，眼中闪过疯狂。

"这就是……雷劫？"吐出带血唾沫，"去你妈的，这就是天道在杀人！"

天道没有给喘息机会。

"第二重天劫：因果剥离——"

乌云中，第二道雷霆凝聚。
这一次，颜色变成诡异银色。不再是符文流，而是一段段锁链般的法则链条。

银色雷霆无声劈落。

这一击，没造成肉体伤害。但银光穿透身体瞬间，感觉到比肉体毁灭更恐怖的剥离感。

记忆在消失。

他与世界的联系正被切断。作为"肖山"的一生——父母笑脸、大学宿舍、送外卖穿过的街道、还有为了救他而消散的沈策……这些画面被银色法则锁链强行从大脑抽出，粉碎。

天地试图在因果层面，将他这个存在彻底抹除。

"给我……滚出去！"

肖山双目赤红，不顾一切催动体内金丹。不懂任何修真功法，唯一的武器就是凡人的意志力。

"我的命，只有我自己能做主！"

金丹爆发太阳般耀眼光芒。金色灵力化作无数无形利刃，在识海疯狂切割，将入侵的银色法则锁链寸寸斩断。

噗——！

喷出一大口鲜血，身体在半空摇摇欲坠。成功保住记忆和因果，但强行对抗天道的代价，是金丹能量开始反噬肉身。皮肤表面出现大面积龟裂，像即将碎裂的瓷娃娃。

天空中，无情审判仍在继续。

"第三重天劫：夺舍覆写——"

纯黑色的雷霆在云层酝酿。吞噬一切光线的黑，代表着天道最深处的抹杀指令。只要被击中，肖山的自我意识将被覆盖，变成绝对服从指令的傀儡。

肖山抬头，看着即将降临的黑色死神。

身体已到达极限。金丹力量庞大，但他无法有效转化，只能被动防御。

"这样下去……必死无疑。"

必须找到破局方法。

目光在绝境中四下搜寻，最终穿过千米高空，落在下方尚未完全被抹除的林江市。

新城区主干道，还有几辆幸存的外卖电动车在崩溃边缘疯狂逃窜。黄色、蓝色的骑手服，在夜风中显得渺小。

外卖电动车。

脑海中，突然闪过一道微弱却足以劈开黑暗的闪电。

死死盯着正在蓄力的黑色雷霆。

作为一个程序员，敏锐捕捉到一个细节：无论是金色雷霆还是银色锁链，在劈落前的蓄力阶段，能量波动呈现出完美的周期性。

从暗到亮，再从亮到暗。

那是一个完美的正弦波（Sine Wave）。

f(t) = A × sin(ωt + φ)

"雷劫的底层逻辑……是正弦波？"肖山眼睛越睁越大，疯狂念头在脑海成型。

既然是波，就有频率。
既然有频率，就可以——共振。

**[第十三章·完]**
