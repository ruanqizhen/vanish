# 第十三章：毁灭之光与跨界雷劫

## 世界的橡皮擦

那种令人作呕的失重感消失的瞬间，肖山并没有像他预想的那样，重重地摔落在坚硬的地面上。

他极其诡异地，悬浮在半空之中。

犹如刀锋般锋利的狂风疯狂地刮过他的脸颊，极其冰冷的空气瞬间灌满了他那几乎要炸裂的肺部。他艰难地低下头，布满血丝的瞳孔在下一秒，瞬间缩成了极细的针尖。

他的脚下，是林江市那璀璨、繁华到了极点的夜景。万家灯火如同倒悬在人间的璀璨星河，纵横交错的街道上车流如织，那些红色的汽车尾灯在黑夜中拉出了一条条长长的、充满活力的光轨。

沈策用那块神秘怀表强行撕开的空间裂缝，并没有将他送往什么未知的异世界，而是直接将他传送到了距离林江市地面上千米的高空。

“沈策……”肖山的心脏就像是被一只无形的铁手死死地攥住，痛得他几乎无法呼吸。他下意识地摸向自己的胸口，隔着破烂的外卖服，他能清晰地感受到那块沈策留给他的、依然散发着冰冷气息的无指针怀表。那个总是冷着脸的男人，用自己的命，将他从那个彻底坍塌的地下空间里硬生生地送了出来。

但是，绝境并没有因为他的逃离而解除。
反而，以一种令人彻底绝望的量级，在整个世界疯狂地扩大。

肖山死死地盯着新城区的方向。在那里，一道完全无法用人类现有的物理学和语言去描述的“光束”，正带着一种摧枯拉朽的恐怖威压，从无尽的九天之上轰然降临。

那根本就不是普通意义上的光。它没有任何的颜色，甚至其存在的维度，都已经远远超越了人类视网膜和大脑能够解析的极限。它就像是一把由神明握在手中的、巨大无比的超级橡皮擦，正在无声无息地，抹过这座繁华的现代都市。

高维的毁灭之光。

那道无色的光束极其冷酷地扫过新城区的一栋四十七层高的标志性摩天大楼。在不到零点三秒的极短时间内，那栋由数万吨钢筋混凝土构成的雄伟建筑，瞬间失去了三维空间赋予它的“厚度”。它就像是被一股不可抗拒的降维打击直接碾过，瞬间被压成了一张没有丝毫厚度的、扁平的纸片，极其荒谬地贴在半空中，就像是一张极其劣质的巨型贴画。

紧接着，这张荒谬的“贴画”开始迅速瓦解。

从边缘开始，扁平的平面开始剥落、崩解成无数只有硬币大小的二维碎片。极其诡异的是，每一块碎片的正面，竟然还完好地保留着那栋大楼在被降维前一瞬间的局部画面——一扇还亮着温馨灯光的窗户、一张惊恐万状的扭曲人脸、半块正在播放着可乐广告的电子屏幕。这些碎片在半空中无助地翻滚着，最终，彻底化为了一种连灰尘都不如的、绝对的虚无。

“它在……抹除整个世界？”肖山浑身冰冷，如坠冰窟。

在这一刻，他终于彻底明白了沈策之前说过的那句“天地法则不允许异数存在”到底是什么意思了。因为他这个极其顽固、连底层抹杀程序都无法处理的系统异数，这个世界的“天道”终于彻底失去了耐心，启动了最高级别的抹杀法则。它根本不打算再去小心翼翼地修补那些因他而产生的系统 BUG，它要用最简单粗暴的方式，直接把这个沙盒世界彻底重置！

下方的街道上，这种恐怖的降维抹除灾难正在以一种疯狂的速度蔓延。

一辆正在主干道上高速行驶的夜班公交车，不幸被那道光束的边缘扫中。它前半截的车身还保持着巨大的物理惯性疯狂地向前冲去，而它的后半截，却已经在瞬间被压成了极其扁平的二维图案。车厢里的乘客甚至连一声惊恐的惨叫都没来得及发出，他们的身体便随着车厢一起被无情地剥离了厚度，变成了一张张定格着极度惊恐表情的纸片人，随后碎裂成了漫天飞舞的虚无方块。

“停下！给我停下！”肖山在千米高空中，对着那片苍穹发出了绝望而凄厉的怒吼，“有什么冲着老子来啊！”

他的吼声瞬间就被高空的狂风撕得粉碎。但是，他刚才被迫吞入体内的那颗极其狂暴的金丹，却在这一刻，产生了一种极其玄妙的感应。

那颗被沈策强行塞进他嘴里的金丹，在经过了最初那阵破坏性的改造后，原本已经安静地悬浮在他的丹田处。但此刻，当肖山的情绪产生了极其剧烈的波动，当他对这座正在被无情屠杀的城市、对沈策的死产生了极其强烈的因果牵绊时，那颗金丹，如同被点燃了引信的核弹，彻底爆发了。

轰——！

一股滚烫到了极点、仿佛能融化一切的金色洪流，直接从他的丹田深处轰然炸开。

这一次，不再是那种撕裂经脉的恐怖剧痛，而是一种浩瀚无垠、纯粹到了极致的庞大能量。金丹的力量沿着他全身的血管和神经疯狂地蔓延、奔涌。他的皮肤表面，开始接连不断地浮现出一道道极其耀眼、深奥难懂的金色符文。这些符文在他的体外迅速交织成了一张极其复杂的因果网络，硬生生地抗住了高空的狂风，将他牢牢地固定在了千米的高空之上。

就在同一秒。
他头顶的天空，骤然变了。

## 杀毒雷劫

在林江市正上方的夜空之中，毫无预兆地，突然凭空出现了一片浓郁得如同泼墨一般的巨大乌云。

那绝对不是什么普通的自然天气变化。在那片翻滚的乌云内部，根本没有任何的水汽，有的，只有密密麻麻、正在以极其疯狂的速度滚动的金色系统符文。那些符文以每秒数万行的恐怖速度在云层中流转、碰撞，就像是某种正在疯狂酝酿的、足以毁灭整个沙盒世界的灭世天劫。

“系统警告：检测到严重破坏平衡的天地异数。当前威胁等级：极度危险。正在请求最高权限……请求通过，即刻启动针对性抹杀法则——”

一阵极其冰冷、毫无任何人类感情波动的机械合成音，没有通过空气传播，而是直接在肖山的意识最深处轰然炸响。

那不是人类的声音。那是这个沙盒世界的“天地法则”，对一个致命病毒下达的无情判决。

天地，终于发现了他。金丹那毫不掩饰的恐怖爆发，让他在这个世界的系统感知中，变得就像是在黑夜中点燃了一颗极其耀眼的太阳，再也无法隐藏。

下方那道正在肆虐的毁灭之光竟然极其诡异地暂停了扫荡。乌云中那些疯狂流转的金色符文开始急速汇聚，最终，极其精准地死死锁定在了肖山的正上方。

“第一重天劫：底层数据清空，形神俱灭。”

咔嚓！

一道足有水桶粗细、蕴含着恐怖威压的金色雷霆，从那片翻滚的云层中，带着毁天灭地的气势，悍然劈落！

那道雷霆通体呈现出一种诡异的半透明状，如果在极度放慢的视野下，甚至能看到那里面流动着亿万行极其细密、代表着绝对毁灭的底层法则代码。它劈落的瞬间，那极其刺目的金光，甚至将整个林江市的夜空照耀得亮如白昼，仿佛将那黑色的天幕硬生生地撕开了一道巨大的裂缝。

肖山根本就来不及做出任何躲避的动作。在这种被天地法则绝对锁定的雷劫面前，任何形式的物理逃避，都彻底失去了意义。

金色的雷霆，极其精准地，狠狠轰击在了肖山的身上。

“呃啊啊啊啊——！”

肖山扬起头，发出了一声凄厉到了极点的非人惨叫。

那根本就不是什么高温的物理电击，而是一种在数据层面极其残忍的“强制删除”！在雷霆触及他皮肤的那一瞬间，他的右半边肩膀，竟然直接凭空消失了！没有鲜血喷涌，没有肌肉焦黑的糊味，他的肌肉、血管、甚至是坚硬的骨骼，就像是被一只无形的超级巨手，在物理规则的层面上，生生地抹去了一块，直接化作了极其微小的光点，消散在了空气中。

那种痛苦，已经彻底超越了人类神经系统能够承受的极限。

但就在下一个瞬间，他体内那颗疯狂旋转的金丹，爆发出了极其强悍的生命力。一股极其磅礴的金色灵力洪流，疯狂地涌向了他那已经消失的右肩。那些正在消散的数据光点，竟然被这股力量硬生生地从虚无中拉扯了回来。他的血肉开始以一种违背生物学常理的方式重新交织、生长，白森森的骨骼肉眼可见地开始再生、缝合。

不到短短的三秒钟，他那被彻底抹除的右肩，竟然奇迹般地恢复如初，连一丝疤痕都没有留下。

肖山大口大口地喘着粗气，浑身早已被冷汗湿透，额头上的青筋如蚯蚓般根根暴起。他死死盯着自己完好无损的肩膀，那双布满血丝的眼中，闪过了一丝极其疯狂的戾气。

“这就是……所谓的雷劫？”他极其凶狠地吐出了一口混杂着血丝的唾沫，“去你妈的修仙，这就是这个操蛋的天道系统在执行杀毒程序！”

但那高高在上的天道，显然没有给他任何喘息和思考的机会。

“第二重天劫：因果节点剥离——”

那片翻滚的乌云中，第二道更为恐怖的雷霆，正在迅速凝聚。
而这一次，雷霆的颜色，变成了一种极其诡异、冰冷的银白色。那里面流淌的不再是单纯的符文数据流，而是一段段如同实质般、代表着因果法则的银色锁链。

银色的雷霆，没有发出任何声音，极其无声无息地劈落了下来。

这一击，并没有给肖山的肉体造成任何实质性的伤害。但当那银光穿透他身体的瞬间，他却感觉到了一种比肉体被抹除还要恐怖一万倍的极度剥离感。

他的记忆，正在被系统强制格式化。

他与这个世界、与他过往人生的所有联系，正在被一根一根地强行切断。作为“肖山”这短短二十多年的一生——父母那早已模糊的笑脸、大学宿舍里键盘的敲击声、那些年送外卖时穿过的每一条肮脏的街道、还有不久前……那个为了救他而甘愿化作飞灰的沈策……这些构成他“存在”的珍贵画面，正在被那些银色的法则锁链，极其强硬地从他的大脑海马体中抽出，然后，无情地粉碎。

天道试图在因果的源头层面上，将他这个“病毒”，彻底、干净地抹除掉，让他变成一个从来没有在这个世界上存在过的虚无！

“给我……滚出老子的脑子！”

肖山双目赤红如血，他不顾一切地疯狂催动体内的那颗金丹。他不懂任何那些玄之又玄的修真功法，他现在唯一能够倚仗的武器，就是他作为一个凡人、一个曾经永不屈服的底层架构师那坚不可摧的意志力。

“老子命，老子的记忆，只有老子自己能做主！”

金丹在他的体内爆发出了一阵犹如超新星爆发般耀眼的光芒。极其纯粹的金色灵力化作了无数把无形、锋利的利刃，在他的识海中展开了疯狂的切割，将那些试图入侵、篡改他记忆的银色法则锁链，寸寸斩断！

噗——！

肖山猛地喷出了一大口鲜血，他的身体在狂风肆虐的半空中摇摇欲坠。他成功地保住了自己的记忆和因果，但作为一个凡人，强行对抗天道法则的代价是极其惨痛的。金丹那庞大的能量，因为没有正确的引导，开始疯狂地反噬他的肉身。他的皮肤表面开始出现了大面积、如同蜘蛛网般的恐怖龟裂，整个人就像是一个即将彻底碎裂的劣质瓷娃娃。

而天空中，那场无情的杀毒审判，仍在继续。

“第三重天劫：底层逻辑夺舍覆写——”

一道纯黑色的雷霆，开始在最厚重的云层深处缓慢地酝酿。那种纯粹的黑，仿佛能吞噬一切光线，代表着天道系统最深处、最高级别的强制抹杀指令。只要被那道黑色雷霆击中，肖山的自我意识将会在瞬间被系统强行覆盖，他将会变成一个没有思想、没有记忆、绝对服从系统指令的一具傀儡空壳。

肖山极其艰难地抬起头，看着那道即将降临的黑色死神。

他的肉体，已经到达了崩溃的绝对极限。虽然他体内金丹的能量依然庞大，但他根本无法将这股力量转化为有效的攻击，只能陷入这种绝望的被动防御之中。

“如果再这样被动挨打下去……我必死无疑。”

必须找到破局的方法。必须在这不可能中，找出系统的破绽。

他在绝境中，目光如鹰隼般四下疯狂搜寻。最终，他的视线穿过了千米的高空，极其精准地落在了下方那座尚未完全被毁灭之光抹除的林江市上。

在新城区那条已经残破不堪的主干道上，竟然还有几辆幸存的外卖电动车，正在世界崩溃的边缘疯狂地逃窜。那些穿着黄色、蓝色制服的骑手，在狂暴的夜风中显得如此的渺小、无助，却又充满了令人动容的求生欲。

外卖电动车。

在看到那些电动车的瞬间，肖山的大脑深处，突然闪过了一道极其微弱、但却足以劈开无尽黑暗的灵感闪电。

他猛地抬起头，死死地盯着云层中那道正在缓慢蓄力的黑色雷霆。

作为一个曾经对各种底层算法极其敏感的架构师，他极其敏锐地捕捉到了一个刚才被他忽略的致命细节：不管是第一道的金色雷霆，还是第二道的银色锁链，在它们劈落前的那段短暂的蓄力阶段，它们散发出的能量波动，竟然都呈现出了一种极其完美的周期性变化。

从能量的极暗到极亮，再从极亮衰减到极暗。

那是一个，在数学模型中极其完美的——正弦波（Sine Wave）。

f(t) = A × sin(ωt + φ)

“这个狗屁雷劫的底层运行逻辑……竟然是一个正弦波函数？”肖山的眼睛越睁越大，一个极其疯狂、甚至可以说是异想天开的念头，在他的脑海中迅速成型。

既然它的本质是一种能量波，那么它就必然存在一个固定的振动频率。
既然它有频率，那么在这个物理规则还没有完全崩溃的世界里，就可以利用一种最基础的物理现象——共振。

**[第十三章·完]**
